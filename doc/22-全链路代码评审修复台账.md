## B1 · 部成单被撤后无限重撤 + 死量预算超额承诺（✅ 已修并测）

**根因**：`PART_TRADED` 同属 `OrderState.terminal()` 与 `active()`，且 `local_ledger.sync_status` 把「`CANCELLED/REJECTED` 且 `filled>0`」的部成撤单收口回 `PART_TRADED`——它仍在 `on_ttl_expired` 可撤集合 `{SUBMITTED,REPORTED,PART_TRADED}` 内，每个 grace 周期被反复撤单；其死量 `remaining` 仍被 `committed_amount` 计入预算承诺。

**设计**：新增独立终态 `OrderState.PART_CANCELLED`（部成单未成部分被撤/废后的终态收口），与「部成仍在途」的 `PART_TRADED` 严格区分。它：① 不在 `on_ttl_expired` 可撤集合 / `rebuild_runtime_state` in_flight 集合内 → 退出 TTL 盯防、不再重撤；② `committed_*` 对它置 `remaining=0` → 死量不占承诺；③ 仍在 `active()`（保 find_active 幂等防重复下单）、`filled>0` 仍占建仓只数名额；④ 三处「不回退」守卫（`sync_status`/`add_fill`/`reconcile_fills_from_detail`）粘滞终态，杜绝迟到回报把它降级回在途态复活重撤循环。

**改动**：
- `contracts/enums.py`：新增 `PART_CANCELLED`，纳入 `terminal()` 与 `active()`。
- `order/local_ledger.py`：`sync_status` 收口改 `PART_CANCELLED` + 不回退守卫扩到 `(TRADED, PART_CANCELLED)`；`add_fill` 状态推进守卫同扩（迟到成交仍累计 filled 但不改终态）；`reconcile_fills_from_detail` 跳过重算守卫 + 收口同步改。
- `order/order_executor.py`：`committed_amount`/`_committed_for_code`/`_committed_remaining_for_code` 对 `PART_CANCELLED` 置 `remaining=0`。
- `data_writer/callbacks.py`：同步过时注释（PART_TRADED→PART_CANCELLED；该函数判据走券商回报状态，逻辑无变化）。

**测试**（新增 4 个，更新 1 个）：
- `test_local_ledger.py`：`test_sync_status_fill_aware_cancel_settles_part_cancelled`（收口 PART_CANCELLED）、`test_part_cancelled_is_sticky_terminal_no_downgrade`（迟到委托态/成交不降级、filled 仍累计）。
- `test_order_executor.py`：`test_part_cancelled_no_recancel_loop`（撤后再 sweep 不复发撤单、退出 TTL 盯防）、`test_committed_amount_excludes_part_cancelled_dead_remaining`（死量 remaining 不计承诺）。

**review**：独立对抗式审查通过——逐处核对所有读 `OrderState.PART_TRADED` 的分支（`on_ttl_expired`/`rebuild_runtime_state`/`_is_position_committed`/对账/持仓机/存储/回调），确认无漏处理、无回归；SELL 方向（`committed_*` 已过滤 `side==BUY`，持仓机不读 OrderState、on_road 回扣按券商回报状态）不受影响；`state TEXT` 无 CHECK 约束、`OrderState(value)` 可还原新值，无需迁移。审查唯一发现 = `callbacks.py` 两处过时注释，已修。

**结果**：全量 **705 passed**（净增 3 用例）。

**SQLite 兼容**：`local_order_ledger.state` 为 `TEXT NOT NULL` 无 CHECK 约束，`mappers.row_to_ledger` 用 `OrderState(value)` 还原，新值跨重启/落盘正常，无需 Alembic 迁移。
