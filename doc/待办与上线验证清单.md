1. **信号侧跑数据库迁移**（F3 新增两列）：
   ```bash
   cd .../stock-ah-premium-ai/backend && . .venv/bin/activate && alembic upgrade head
   ```

2. **执行侧提供交易日清单文件**（E1 fail-closed 需要）：
   ```bash
   # 信号侧导出（脚本 backend/scripts/export_trade_calendar.py）
   python scripts/export_trade_calendar.py --output /opt/qmt/trade_days.txt --min-future-days 90
   # scp 到 Windows 交易机后配：
   QMT_TRADE_CALENDAR_FILE=C:\qmt\trade_days.txt
   ```
   未配且未显式放行 → 进程启动期 RuntimeError 拒绝启动。每年节假日安排公布后重新导出 scp。
3. **资金/风控关键开关**（建议显式配，详见 README §8.4）：`QMT_TARGET_POSITION_RATIO`、`QMT_PER_ORDER_MAX_AMOUNT`、`QMT_MAX_POSITION_PER_STOCK`、`QMT_MAX_TOTAL_EXPOSURE`、`QMT_ACCOUNT_DRAWDOWN_LIMIT`/`QMT_STOCK_FLOAT_LOSS_LIMIT` 等。
