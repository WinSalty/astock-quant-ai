## 〇、制度与术语基线

> 本节是全手册交易制度与高频术语的唯一权威口径，后续各章只列与本节不同的差异点。文中所有数值阈值均为经验区间，须结合个人样本回测校准；交易制度细节以交易软件与交易所最新公告实时显示为准。

### 0.1 涨跌幅与板块

- 沪深主板（600/601/603/605、000/001/002）：±10%。
- 创业板（300/301）、科创板（688）：±20%（俗称 20cm）。
- 北交所（8xx/43x/83x/87x）：±30%（俗称 30cm）。
- **ST/\*ST**：主板风险警示股截至本手册撰写日（2026-06-13）仍执行 **±5%**；沪深北交易所已发布修订规则，自 **2026-07-06 起主板风险警示股涨跌幅由 ±5% 调整为 ±10%**；创业板/科创板风险警示股为 ±20%。切换日前后以券商行情实际显示的涨跌停价为准。**本战法原则上回避 ST 类标的**（退市风险高、流动性差）。

### 0.2 新股与无涨跌幅阶段

- 主板/创业板/科创板新股上市前 5 个交易日不设涨跌幅，第 6 日起恢复对应板块限制；北交所新股上市首日不设涨跌幅、次日起 ±30%。
- **30%/60% 两档盘中临时停牌**（每次 10 分钟）只适用于"无涨跌幅限制"的股票（新股前 5 日、北交所新股首日），**不是北交所或任何已上市股票的日常盘中机制**。
- 无涨跌幅阶段不存在"涨停板"，不属于本战法打板范畴。

### 0.3 交易时间与集合竞价

- **早盘集合竞价 9:15–9:25**：9:15–9:20 可撤单，9:20–9:25 不可撤单；9:25 撮合产生开盘价；9:25–9:30 只接收申报、不撮合（盘前静默期）；9:30 起连续竞价。
- **收盘集合竞价 14:57–15:00**：沪深各板块（含沪市主板）均适用，全程不可撤单、不接受市价申报。
- **T+1**：当日买入次日方可卖出。打板的盈亏全部押在"次日开盘到尾盘"这一不可逃逸窗口，隔夜风险（消息面、大盘、情绪退潮）不可控。

### 0.4 价格笼子（连续竞价阶段）

- 主板、创业板：买入申报价不得高于"基准价×102% 与 基准价+10 个最小变动单位"的**孰高值**；卖出申报价不得低于"基准价×98% 与 基准价−10 个最小变动单位"的**孰低值**。瞬时极端价位的扫单因此受限，追板挂单须理解该撮合约束。

### 0.5 核心指标统一口径

- **炸板率**：<30% 资金积极、30%–50% 分歧加剧、>50% 退潮确认。
- **晋级率**（两个不同分母，勿混用）：① 全市场"首进二"晋级率 >40% 偏健康、<25% 衰竭；② 主线/板块连板晋级率 连续 2 日 ≥50% 确认延续、跌破 50% 分化。
- **换手率**（两个口径，勿混用）：① 首板健康换手 25%–40% 为佳，<10%~15% 偏低（缩量一字除外），>50% 才算筹码过度松动；② 连板加速段龙头日均换手 10%–30%（封单锁仓时换手反而下降）。
- **封单强度**：封流比 = 封单金额 ÷ 流通市值，≥2% 封板较实、1%–2% 一般、<1% 偏弱。
- **单票仓位上限**（占账户本金，按板幅）：10cm ≤20%、20cm ≤10%、30cm ≤7%（对应单笔账户最大风险约 2%）；总仓位随情绪周期调节，退潮/冰点空仓。

### 0.6 关键角色术语

- **龙头**：板块或全市场连板最高、人气最旺、封板最快、跟风最广、流动性最好的核心标的，享有最高容错与最大弹性。
- **中军**：体量较大、跟随龙头的板块第二梯队，常用于验证主线资金强度。
- **补涨**：龙头与中军之后才启动的同题材个股，弹性与容错均弱于龙头。
- **卡位**：在题材中提前占据有利身位（板数/位置）的个股。
- **杂毛**：跟风、概念牵强、流动性差的尾部跟风股，本战法明确剔除。

## 一、战法定位与使用纪律

### 1. 战法定位：你在赚谁的钱

本战法是一套**纯情绪驱动的超短线接力体系**，不依赖业绩、估值或中长期逻辑。它只做一件事：在市场情绪上行周期里，识别并买入当下**最强势的人气龙头**，赚取**情绪溢价**与**资金合力**的钱，然后在情绪退潮或合力瓦解前离场。

- **核心思想**：顺应市场情绪，与市场合力同向，只买"被市场选出来的最强者"，遵循"强者恒强"。龙头不是事先预测出来的，而是被场内资金用真金白银打出来的——它通常具备最高换手、最快封板、最大成交、最广跟风的特征。
- **你赚的不是公司的钱，是博弈对手的钱**：超短资金的盈利来自后续接力资金给出的溢价。当一只票成为人气龙头，场内资金对其形成做多合力，封单堆叠、隔日溢价兑现，先入场者把筹码以更高价交给后入场者。这部分溢价就是"情绪溢价"，它随情绪周期生灭，**不具备基本面支撑，不可长期持有**。
- **为什么只做龙头**：龙头有"多条命"——退潮时它最抗跌（容错高），起涨时它最先反包（弹性大），且全程流动性最好（想走能走掉）。杂毛（跟风补涨股）盈亏比差、流动性差、退潮首当其冲，是本战法明确剔除的对象。

### 2. 适用与不适用的市场环境

战法的有效性**完全取决于市场情绪所处的周期位置**。判断环境是开仓的前置条件，环境不对时空仓本身就是策略。

**适用环境（满足越多越好，至少满足前两条才考虑进场）：**

- 情绪处于**回暖期或主升期**：连板高度梯队完整（存在 3 板及以上的领涨者），且高度在抬升而非坍塌。
- **赚钱效应为正**：当日涨停家数明显多于跌停家数，昨日涨停今日红盘比例（晋级率/溢价）为正。其中"涨停数 ≥ 50 家且封板率 > 60%"是**发酵期及以上的进攻门槛**；启动期涨停数回到 30–50 家时只可小仓试错，不视为重仓信号（与第二章启动期口径衔接）。
- 有**清晰的主线题材**，资金向少数方向集中，而非散乱在多个无延续性的概念上。
- 大盘指数无系统性下杀风险（无重大利空、无连续放量阴线破位）。

**明确不适用 / 必须空仓的场景：**

- **情绪冰点 / 退潮期**：高度板批量炸板、连板梯队断层、昨日涨停今日大面，赚钱效应转负（亏钱效应主导）。此时任何"抄底龙头"都是接刀。
- **无主线的震荡市 / 缩量阴跌市**：题材散乱、无延续性，龙头打出来第二天就死，接力无人承接。
- **系统性风险释放期**：指数级别破位下跌、政策性利空、外围黑天鹅。情绪溢价在普跌中归零。
- **个股已处于高位高度板的极端分歧/一致**：参与门槛与容错远超普通交易者承受范围（见高风险说明）。
- **流动性不足的标的**：日成交额过小、封单结构脆弱、容易"封不住又走不掉"的票，不具备龙头资格。

### 3. 硬性交易纪律（违反任意一条即视为操作失败，不论盈亏）

以下纪律是战法的生存底线，不是建议。盈利的前提是先活下来。

1. **计划交易，交易计划**：任何买入前必须明确写下——标的、买入触发条件、买入价位区间、最大仓位、止损价、次日卖出预案（分歧走人 / 一致留板）。**没有预案不开仓，盘中不临时改计划。**
2. **只做龙头，不做杂毛**：只参与梯队最高、人气最旺、封板最快、跟风最广的那一个（或两个）核心标的。补涨股、跟风股、"便宜的同概念"一律不碰。
3. **绝不亏损补仓 / 摊低成本**：持仓走坏时唯一动作是止损，禁止向下补仓。补仓是把一次错误放大成致命错误的最快路径。
4. **严守止损，先于一切**：止损价在买入时就已设定，触发即无条件执行，不看成本、不抱幻想、不等反弹。打板买入后若**盘中开板回落、封单快速消失（炸板）**，按预案处理；次日**核按钮/大幅低开破止损位**，开盘集合竞价或开盘后立即止损出局。
5. **控制交易频率与情绪**：机会稀缺，**一周开仓不超过 2 次**为宜；宁可错过，不可做错。不追、不赌、不报复性交易；连续两笔亏损当日强制停手。
6. **控制单票与总仓位**：单一标的按板幅控制（10cm 标的 ≤20%、20cm 标的 ≤10%、30cm 标的 ≤7%，对应单笔账户最大风险约 2%，详见《制度与术语基线》/第十三章）；总仓位随情绪周期调节，周期不明朗时降至 3 成以下，退潮/冰点空仓。
7. **回撤后强制降频**：当账户从近期高点回撤达到预设阈值（建议 **-8% 触发降频、-15% 触发停手**），立即将仓位与开仓频率减半甚至清零，先复盘找回手感与情绪，再逐步恢复。**回撤期加码是账户归零的标准剧本。**

### 4. 高风险属性的客观说明

打板/追板是 A 股风险等级最高的交易方式之一，参与前必须清楚其结构性风险，本手册不承诺任何胜率，也不诱导交易。

- **盈亏极不对称且波动剧烈**：在涨跌幅 沪深主板 ±10% / 创业板（300/301）、科创板（688）±20% / 北交所（8xx/4xx）±30% 的制度下，一次封板失败的隔日大面，单笔回撤可达涨跌幅上限附近。**T+1 制度**下当日买入次日才能卖出，意味着打板后必须**承担一整夜的隔夜不可控风险**（消息面、大盘、情绪退潮）。
- **追高即承担最大回撤**：打板买在当日最高价区域，一旦次日不及预期，亏损从最高点起算，容错极低。
- **流动性会在最需要时消失**：连板票的卖出依赖封单与人气，退潮时可能"想卖卖不掉"（一字跌停、封死跌停），止损未必能在理想价位成交。
- **价格笼子约束追单**：主板、创业板连续竞价阶段实行价格笼子（买入申报价不得高于"基准价×102% 与 基准价+10个最小变动单位 的孰高值"，卖出不得低于"基准价×98% 与 基准价−10个最小变动单位 的孰低值"），瞬时极端价位的"天地板"式扫单被限制，追板需理解其撮合规则。
- **集合竞价是关键风险窗口**：早盘 **9:15–9:25** 开盘集合竞价（其中 9:15–9:20 可撤单、9:20–9:25 不可撤单），9:25 撮合产生开盘价，9:25–9:30 只接收申报不撮合（盘前静默期），9:30 起连续竞价；该窗口反映隔夜多空预期，是打板核按钮的主要发生窗口。尾盘 **14:57–15:00** 收盘集合竞价，沪深各板块（含沪市主板）均适用、全程不可撤单、不接受市价申报，决定收盘结构。竞价高开抢筹与竞价砸盘出货，都集中在这两个时段。
- **制度处于动态调整中，须实时核对**（基于当前注册制框架，截至 2026-06-13）：
  - 新股上市：主板/创业板/科创板新股上市**前 5 个交易日不设涨跌幅限制**，第 6 个交易日起恢复对应板块限制（主板 ±10%、创业板/科创板 ±20%）；北交所新股上市**首日不设涨跌幅**、次日起 ±30%。
  - 30%/60% 两档盘中临时停牌（每次 10 分钟）**只适用于"无涨跌幅限制"的股票**（新股前 5 日、北交所新股首日），不是北交所或任何已上市股票的日常盘中机制。
  - **ST/*ST 股票截至本手册撰写日（2026-06-13）仍执行 ±5%** 涨跌幅；沪深北交易所已发布修订规则，自 **2026-07-06 起主板风险警示股涨跌幅由 ±5% 调整为 ±10%**，创业板/科创板风险警示股为 ±20%。切换日前后以券商行情实际显示的涨跌停价为准，**本战法原则上回避 ST 类标的。**

### 5. 本手册的使用方式

- **按情绪周期判断，再谈个股**：先用本章"适用/不适用环境"过滤当日是否可参与，**环境不对一律空仓**；环境合格后，再进入后续章节的选股、买点、卖点流程。顺序不可颠倒。
- **纪律优先于技巧**：本章第 3 节的硬性纪律是所有后续战法（低吸、半路、打板、接力）的共同前置约束。任何买卖法在违反纪律时一律作废。
- **量化执行、留痕复盘**：每一笔交易都应有可回溯的计划与触发条件，盘后逐笔复盘"是否按计划执行"，而非只看盈亏。**评判操作对错的标准是纪律执行度，不是单笔结果。**
- **先轻仓验证再加码**：在掌握情绪周期判断之前，建议以最小仓位（或模拟）跑通完整流程，确认自身能稳定执行止损与降频后，再逐步放大。
- 后续各章将分别展开：情绪周期与赚钱效应量化判断、龙头识别标准、打板/半路/低吸的具体买点与封单条件、次日分歧/一致的卖点决策、仓位与资金管理细则。本章只确立**定位与纪律**，不与后续章节的具体操作重复。


## 二、市场情绪周期与节奏判断

短线游资的本质不是研究公司，而是研究"人在亏钱效应和赚钱效应之间如何切换"。同一套打板动作，放在情绪不同阶段，期望收益可以从正变负。因此**先判断情绪所处阶段、再决定是否出手与出手力度**，是优先级高于选股本身的一步。本章给出可量化的阶段刻画、温度指标与对应的仓位/打板纪律。

> （交易制度统一口径见《制度与术语基线》一节，本章只列差异点）本章差异点：ST/\*ST 现行 ±5%，但自 **2026-07-06 起主板风险警示股上调至 ±10%**（创业板/科创板风险警示股为 ±20%），切换日前后以券商行情实际显示的涨跌停价为准，届时 ST 板的"封单含金量"与连板节奏会被重估，相关阈值需同步调整；本战法原则上回避 ST 标的。本章所有阈值是**经验区间**，需结合自身样本动态校准，不是机械信号。

### （一）情绪周期的六个阶段：先定性，再定量

情绪周期不是日历周期，而是由"最高连板高度"这条主线串起来的资金行为序列。一个完整周期的标准顺序是：**冰点 → 启动 → 发酵 → 高潮 → 分歧 → 退潮 → （回到冰点）**。判断阶段的第一抓手永远是**当前市场最高连板的高度方向（在升还是在降）**，其次才是配套指标。

- **冰点期**：上一轮退潮已经走完，亏钱效应释放充分。特征是最高板被压到 2–3 板甚至无连板，涨停家数常 < 30 家、跌停家数仍偏多，所有题材"一日游"、隔日普遍低开杀跌，红盘率极低。市场情绪麻木、看谁拉板谁就被砸。**机会属性**：右侧未到，但孕育下一轮启动的反包/首板就藏在这里。
- **启动期**：在冰点之后，活跃资金开始试探高度，出现 4 板级别的独苗，但连板个股数量少（常 < 10 只）、主线尚不明确。特征是涨停家数回到 30–60 家、炸板率回落到 30% 一线、跌停家数 < 10 家、昨日涨停今日开始有正溢价。**机会属性**：试错窗口打开，容错率开始上升。
- **发酵期**：高度被打开后题材扩散，赚钱效应外溢到首板和低位补涨，连板梯队（2 板→3 板→4 板）开始连续、人气从龙头向跟风传导。涨停家数放大、炸板率维持低位、晋级率明显抬升。**机会属性**：周期内胜率最高、容错最好的主升段。
- **高潮期**：市场选出明确主线与龙头，走出一致性加速，最高板冲到 5 板及以上、涨停家数常 > 60 家、跌停几乎消失、封板资金（封单金额）巨大、隔日溢价普遍 ≥ 3%。表面最舒服，**但这是赔率最差的位置**——情绪已被price in，接下来大概率是分歧。
- **分歧期**：龙头出现第一次显著分歧（盘中开板、或首次收阴/未封住），高位个股内部开始分化：龙头还在但中军、卡位、补涨开始杀跌。指标上表现为连板"断层"（高位独苗存活、低位梯队稀薄）、炸板率抬到 30%–50%、跌停家数升到 10–30 家、隔日溢价转负。分歧若被快速修复（次日龙头重新一致），周期延续；若修复失败，直接转退潮。**机会属性**：只做"分歧转一致"，不做"分歧转退潮"，是本阶段唯一可执行逻辑。
- **退潮期**：分歧未修复，亏钱效应快速扩散。典型形态是"龙头还在加速、中位集体闷杀"，随后龙头通常在 1–2 个交易日内开板/断板。指标上炸板率 > 50%、涨停家数 < 30 家、跌停家数 > 30 家、连板梯队基本消失、昨日涨停今日大面积低开杀跌、溢价 ≤ -5%。**机会属性**：空仓为最优解；退潮期与冰点期是短线全周期里赚钱效应最差的两段，绝大多数亏损发生在"没看到撤退信号"或"在退潮里反复换题材试错"。

判断阶段时，**以连板高度的方向（升/降）定主轴，用涨停/跌停/炸板/溢价四组指标交叉验证**，避免单指标误判。下表为基线，落地时务必用自己最近 20–40 个交易日的样本回测校准。

| 阶段 | 最高连板 | 涨停家数 | 炸板率 | 跌停家数 | 昨日涨停今日溢价 | 主线状态 |
|---|---|---|---|---|---|---|
| 冰点 | 无连板或 2–3 板被压 | < 30 | 偏高 | 偏多 | 明显为负 | 无、一日游 |
| 启动 | 出现 4 板独苗 | 30–60 | ~30% | < 10 | 由负转平/微正 | 雏形未明 |
| 发酵 | 梯队连续上移 | > 50 放大 | < 30% | < 10 | ≥ 0，逐步走高 | 多主线扩散 |
| 高潮 | 5 板及以上 | > 60 | < 30% | 近 0 | ≥ 3% | 单一主线一致 |
| 分歧 | 高位断层 | 30–60 | 30%–50% | 10–30 | 转负 | 龙头强、中军杀 |
| 退潮 | 连板消失 | < 30 | > 50% | > 30 | ≤ -5% | 全线杀跌 |

### （二）可量化的情绪温度指标体系

把上面的"感觉"拆成可每天复盘的数字。建议固定收盘后采集以下九项，做成自己的"情绪温度表"，连续记录方能识别**拐点**（拐点比绝对值更重要）。

1. **最高连板高度（主轴）**：今日全市场最高几板。**方向 > 数值**——连续抬高=情绪向上，从 N 板跌回 N-2 以下=高度断裂、警惕转折。冰点常无 4 板、高潮常见 5 板+。
2. **连板梯队结构是否健康**：从 2 板到最高板逐级是否"有兵"。健康的梯队是 2 板、3 板、4 板各有数只、逐级递减但不断层；**断层（高位只剩独苗、3 板以下塌陷）是分歧/退潮的早期信号**。梯队健康度比单看最高板更能预判延续性。
3. **首板数量与晋级率（首进二、二进三、三进四…）**：晋级率 = 次日成功晋级数 ÷ 前一日同高度涨停数。
   - **全市场首进二晋级率（口径9①）**：首进二 > 40%、二进三 > 30% 视为偏健康；逐级晋级率系统性抬升=发酵，系统性塌陷（全市场首进二跌破 ~25%）=赚钱效应衰竭。**注意：本指标分母是全市场前一日同高度涨停数，与第三章"板块连板晋级率（连续 2 日 ≥ 50% 确认延续）"分母不同（后者分母是单一板块前一日连板个股数），两者不可混用。**
   - 首板数量本身是"柴火量"：首板多但晋级率低=情绪虚火，追高即埋。
4. **炸板率**（炸板数 ÷（涨停数+炸板数））：< 30% 资金积极、敢封；30%–50% 资金犹豫、分歧加剧；> 50% 追高必套、退潮确认。封板率（1 - 炸板率）≈ 75% 以上为强、≈ 55% 以下为弱可作对照。
5. **涨停家数**：> 60 强、30–60 中性、< 30 弱。需结合炸板率看——涨停多但炸板率高=假强。这是第一章"赚钱效应为正"进攻门槛的来源指标：涨停数 ≥ 50 家且封板率 > 60% 是允许进攻的最低环境基线，本章 > 60 家的"强"档位高于该门槛，是更确定的发酵/高潮特征。
6. **跌停家数（亏钱效应直读）**：< 10 安全、10–30 分歧加剧、> 30 退潮确认。跌停家数放大往往**领先**于涨停家数萎缩，是更灵敏的撤退信号。
7. **封板资金（封单含金量）**：统一用**封流比 = 封单金额 ÷ 流通市值**衡量——≥2% 封板较实、1%–2% 一般、<1% 偏弱。观察龙头与高位板的封流比、是否盘中反复开板。封单厚实且全天不松动=资金一致；封流比低、尾盘才回封、巨量换手才封住=强转弱前兆。（注意：ST 板在 ±5% 时代封单"看着大"是低成交假象，2026-07-06 主板 ST 改 ±10% 后须重新评估。）
8. **昨日涨停今日的红盘率 / 溢价**（最直接的"接力赚钱效应"）：取昨日所有涨停股，统计今日**红盘率**（收红占比）与**平均溢价**（按今日均价或开盘价口径，固定一种）。红盘率 > 50% 且平均溢价 ≥ 0 为可接力；红盘率 < 30%、平均溢价 ≤ -3% 为亏钱效应主导，次日打板期望为负。**这是判断"明天还能不能打板"最务实的单一指标**。
9. **赚钱效应 / 亏钱效应净值**：可用"昨日涨停今日红盘率 − 昨日跌停（或炸板）今日表现"或自定义加权综合分，量化净赚钱效应。连续 2 日由正转负是减仓信号。

**采集口径提醒**：所有比率类指标必须固定剔除/包含 ST、新股次新（上市前 5 日无涨跌幅、不计入连板高度统计）、北交所（±30% 节奏与主板不可比，建议单列）的规则，否则跨日不可比。一字板、换手板要分开看——一字板封单大不等于资金强，可能只是无人卖。

### （三）大盘与指数环境判断

情绪温度指标刻画的是"连板生态"本身，但连板生态嫁接在大盘指数环境之上：指数级破位下杀会同时抽走所有题材的承接资金，使温度指标"看着还行却突然失效"。本小节给出可每日复盘的指数级量化口径，与情绪温度表并列使用、互相印证。

- **跟踪指数清单（固定每日采集）**：以**上证指数**看主板权重与系统性风险、**创业板指**看成长/高弹性资金温度、**Wind 全A（或万得全A）**看全市场真实重心；辅助看**微盘股指数**（小盘投机资金活跃度，与连板情绪同源）与**连板指数 / 涨停板指数**（直接反映打板群体的盈亏曲线）。连板指数走平或破位，往往比大盘指数更早预警短线生态恶化。
- **关键均线破位定义**：以**收盘价跌破 20 日均线（短期生命线）或 60 日均线（中期趋势线）且次日不能收回**为破位确认信号——盘中跌破、收盘收回不算破位，连续 2 个交易日站不回对应均线才视为有效破位。上证指数/创业板指任一出现 60 日线有效破位，视为系统性风险升级。
- **单日放量阴线倍数阈值**：当日成交量较**前 5 日均量放大 ≥ 1.5 倍**、同时收**实体阴线并伴随关键均线（20/60 日）破位**时，定义为"放量破位阴线"，是机构资金撤离的量价信号，出现即降仓。若放量阴线未破均线，先记录、观察次日是否补跌。
- **连续阴线根数预警**：指数**连续 3 根阴线**（无论放量与否）进入预警；**连续 3 根放量阴线**或**5 个交易日内 4 阴**视为趋势转弱确认，短线环境从"可进攻"降为"只防守"。
- **指数与连板高度背离的降仓/空仓规则**：当**连板高度仍在抬升、但大盘指数或连板指数已出现上述破位/放量阴线/连续阴线信号**时，属于"情绪局部火热、资金面整体抽水"的背离，按以下口径降档——出现任一背离信号：总仓位**减半**并停打 2 板以上追板；出现两条及以上背离信号（如 60 日线有效破位 + 放量破位阴线）：**清仓或仅留观察仓**，等待指数重新站回均线、放量阴线被次日阳线吞没后再恢复进攻。指数环境与连板温度只要一边转空，就**以更保守的一边为准**，不用连板的热度去对抗指数的破位。

### （四）冰点反包 / 启动机会的识别

冰点和启动期赔率最高、但右侧信号最弱，**核心是等"第一个能扛住的标的"用涨停给出右侧证据，而不是猜底**。检查清单（需同时满足多条，宁可错过不可抢跑）：

- **首阴反包 / 弱转强**：前期人气股或卡位股在情绪最差时该弱不弱——预期低开杀跌却被资金异动拉起、放量收复分时均价线并最终封板，是冰点出现资金的第一证据。买点是"封板瞬间或确认回封"，不是盘中下跌途中的猜底接刀。
- **跌停家数先于涨停家数改善**：跌停从 > 30 快速缩到 < 15，是亏钱效应释放完毕的领先信号，比涨停家数回升更早出现。
- **昨日涨停今日红盘率拐头**：从 < 30% 抬到 50% 上方且持续 1–2 日，说明接力赚钱效应回归。
- **出现 4 板独苗且未被砸死**：启动期标志。独苗能延续、且次日带动首板/低位补涨放量，才确认"高度被资金认可"。
- **题材从"一日游"变"隔日仍有溢价"**：同一题材连续 2 日有正溢价，是发酵的前置条件。

**冰点期的纪律**：右侧未确认前用最小试错仓（≤ 1–2 成）只打"弱转强首板/反包"，不打高位、不追二板以上；信号被证伪（首板次日低开破板、红盘率不回升）立即止损出局，不加仓、不补仓、不"等反弹"。冰点反包失败率本就高于成功率，靠的是**小仓试错 + 严格止损**控制单次亏损，而非提高单次胜率。

### （五）高潮转退潮的撤退信号

亏损主要发生在这里。**情绪最舒服、隔日溢价最高、人最满仓的时候，正是赔率最差的时候**。出现以下任一信号即应减仓，出现两条及以上立即清仓，不要为已实现盈利"再赌一把"：

- **最高板高度断裂**：从 5+ 板回落且第二天无人接力新高，高度梯队出现断层（高位独苗、低位塌陷）。
- **龙头第一次显著分歧未被次日修复**：龙头盘中反复开板/尾盘才回封/首次收阴，且次日不能重新一致——这是"分歧转退潮"而非"分歧转一致"。龙头首阴/断板后，通常 1–2 个交易日内亏钱效应全面扩散。
- **中军、卡位、补涨集体杀跌而龙头还在硬撑**："龙头加速、中位闷杀"是退潮最经典的盘面，龙头的最后一冲（包括断板后的**反包回封**）多是出货机会而非买点。
- **炸板率跳升到 50% 以上、跌停家数升到 30 家以上**：资金已不敢封板，封板的也封不住。
- **昨日涨停今日大面积低开、红盘率跌破 30%、平均溢价 ≤ -3%～-5%**：接力赚钱效应消失，次日打板期望转负，是最直接的"停手"指令。
- **封单含金量恶化**：高位板封流比变低、需巨量换手才封、尾盘才回封；强转弱的标志是"该强不强"。

执行层面：**减仓优先减高位高标，分歧/退潮中宁可隔日竞价直接出，也不要在连续跌停里等流动性**（T+1 下退潮第二天集合竞价的流动性往往最后一次较好）。撤退是机械动作，不依赖判断"会不会反弹"。

### （六）各阶段的"能否打板 / 打几板 / 仓位"判断条件

下表把阶段判断落到可执行口径。**仓位为总仓位上限的相对建议，非绝对值**；单只标的的具体止损线、买点（封板买/弱转强买/竞价买）由个人体系决定，此处只给阶段级的"开关与档位"。所有数字需用自身资金体量、滑点与历史样本校准，且任何单票不得突破《制度与术语基线》的单票仓位上限（10cm ≤20%、20cm ≤10%、30cm ≤7%）。

| 阶段 | 能否打板 | 打几板（高度上限） | 总仓位区间 | 触发与纪律 |
|---|---|---|---|---|
| 冰点 | 谨慎、仅试错 | 只做首板 / 弱转强反包 | 0–2 成 | 必须出现"该弱不弱、放量回封"右侧信号才动；证伪即止损，不补仓 |
| 启动 | 可，小步 | 首板为主，至多 2 板 | 2–4 成 | 4 板独苗未被砸死 + 跌停 < 10 + 红盘率回升；做主线方向 |
| 发酵 | 可，主攻 | 可打 2–3 板 | 4–7 成 | 梯队连续 + 晋级率抬升 + 炸板 < 30%；顺最强主线、做中军/卡位 |
| 高潮 | 可但赔率差 | 不追最高板，做低位补涨/接力 | 由 6–7 成逐步降至 3–4 成 | 隔日溢价 ≥ 3% 时**减仓而非加仓**；不打 5 板以上接力 |
| 分歧 | 仅"分歧转一致" | 只做次日重新一致的龙头 | 2–4 成且只打 1 只 | 分歧次日修复才动；修复失败立即归零 |
| 退潮 | 否 | 不打 | 0–1 成（仅留观察仓） | 跌停 > 30 / 炸板 > 50% / 红盘率 < 30% → 空仓，竞价清仓 |

几条贯穿全周期的硬约束：

- **方向决定能不能打，结构决定打几板，温度决定仓位多少**。三者任一恶化就降档，不要用仓位赌方向。
- **最高板高度在降时不打追板**；只有高度在升、梯队健康、晋级率向上时才允许打 2 板以上。
- **昨日涨停今日红盘率/溢价是当天打板的"总闸"**：连续为负就关闸，不论题材多诱人。
- **指数环境与连板温度取保守一边**：本章（三）的指数破位/背离信号一旦触发，即便连板温度仍高也按背离规则降仓或空仓。
- 本章特定新增提示：**ST 标的在 2026-07-06 主板风险警示股由 ±5% 调整为 ±10% 的切换日前后，同一只票的"封单含金量"与单日波动口径会突变**，相关阈值（封流比、炸板率分母）须在切换日重新标定后再据以决策，本战法原则上回避 ST 标的。


## 三、题材主线与板块卡位

短线交易的本质，是在最强势的资金共识方向里，用最优的卡位结构博取超额收益。本章讨论如何从全市场的题材噪声中识别出真正的主线、判断其级别与持续性、读懂板块的梯队结构，并据此完成卡位个股的选择与生命周期各阶段的攻防。（交易制度统一口径见《制度与术语基线》一节，本章只列差异点）与本章直接相关的差异点是：ST/\*ST 风险警示股截至本手册撰写日（2026-06-13）仍执行 ±5%，自 2026-07-06 起主板风险警示股由 ±5% 调整为 ±10%（创业板/科创板风险警示股为 ±20%），切换日前后以券商行情实际显示的涨跌停价为准；本战法原则上回避 ST 标的。

### 3.1 识别市场主线与支线

主线是当下全市场资金共识最集中、赚钱效应最强、能持续吸纳新资金的方向；支线是同期次强、跟随主线情绪外溢、但资金厚度和持续性都不及主线的方向。判断主线不靠主观感觉，靠可观测的资金与赚钱效应指标。

- **量化主线的客观信号（盘后用涨停板复盘工具统计）**：
  - 板块内当日涨停家数 ≥ 5 家，且连续 2 个交易日维持，初步具备主线雏形；涨停家数 ≥ 8 家且含 2 个以上连板个股，主线基本确立。
  - 板块内最高连板高度（“板块高度”）≥ 3 板，且高度由该板块个股领跑全场（即全市场最高板就在这个板块里）。
  - 板块内连板晋级率（次日封板个股数 / 前一日连板个股数）连续 2 日 ≥ 50%，说明资金愿意为该方向接力。
  - 板块成交额占全市场比重显著抬升、龙头个股进入全市场成交额前 20，资金用真金白银投票。
- **主线 vs 支线的区分清单**：
  1. 高度：主线出最高板，支线高度通常比主线低 1–2 板。
  2. 厚度：主线连板梯队完整（见 3.3），支线往往只有 1–2 只独苗，缺中军与梯队。
  3. 资金属性：主线由实力游资、机构席位介入，龙虎榜出现知名游资专用席位与机构买入；支线多为小资金博弈、净额小。
  4. 抗跌性：大盘回调日，主线龙头能率先反包或缩量分歧后再上，支线直接断板退潮。
- **操作含义**：资金和注意力优先给主线龙头与中军；支线只在主线分歧、资金外溢且支线自身出现独立连板时小仓参与，不与主线争夺主仓位。同一时间内，重仓方向不超过 1 条主线 + 1 条支线。

### 3.2 题材级别与持续性

题材的天花板由其“级别”决定。级别越高、逻辑越能落到实际业绩与产业趋势，炒作的持续时间窗口越长、能容纳的资金越大、回撤后越容易反复。按持续性从强到弱大致分四档：

- **产业大趋势（持续数月至数年，可反复发酵）**：有真实产业链放量、订单、渗透率提升支撑（如算力/AI、半导体国产替代、新能源细分、机器人产业链）。特征是板块会经历多轮“发酵—退潮—再发酵”，每轮都有新龙头。操作上可做波段+打板结合，退潮后保持跟踪等待二波。
- **政策驱动（持续数周至数月）**：由顶层文件、部委规划、重大会议催化（如新型工业化、国资改革、特定区域规划）。持续性取决于后续是否有配套细则与资金落地。第一波政策预期最猛，细则落地常是“利好兑现”的分歧点。
- **事件驱动（持续数日至两周）**：突发地缘、自然灾害、单一事件、概念性催化（如某主题展会、单一公告）。持续性短，常是 3–5 个交易日的脉冲，高潮即顶，缺乏第二波承接。
- **业绩驱动（季报/年报窗口，与财报周期绑定）**：由实际业绩超预期驱动，逻辑最硬但情绪弹性相对小，更适合“低吸+趋势”而非极致打板；预增公告日的脉冲若无主线配合，常一日游。
- **级别评估清单（动手前逐条打分）**：
  1. 逻辑是否能落到 2–3 家公司的真实订单/产能/业绩？能→加分。
  2. 是否有官方/权威信源（政策文件、行业数据、公司公告）背书，而非小作文？有→加分。
  3. 是否处在产业链放量的早中期（渗透率 < 30%）而非尾声？是→加分。
  4. 此前是否已被反复爆炒、估值透支、套牢盘厚重？是→减分。
- **持续性与操作窗口的映射**：产业大趋势容忍“做长一点”，事件驱动必须“快进快出、高潮即走”。级别越低，越要把止盈触发点前移、把容错时间窗压缩到 1–3 个交易日。

### 3.3 板块联动、板块高度与梯队结构

一个健康的主线板块，应呈现完整的梯队结构与有序联动。梯队结构既是确认主线强度的工具，也是卡位选股的地图。

- **四级梯队定义与角色**：
  - **龙头（龙一/龙二）**：板块最高连板、涨幅与成交额双居前、定义板块高度与空间。龙头的封板强度（封流比 = 封单金额 ÷ 流通市值，是否一字/秒板）是全板块情绪的锚。
  - **中军**：流通市值较大（常见 100–500 亿）、跟随龙头但波动相对稳健的核心股，往往是机构与大游资共同认可的“逻辑最正”标的。中军走稳，板块才有厚度；中军放量上涨是主线升级信号，中军断板是主线见顶的早期警报。
  - **补涨**：龙头、中军走出空间后，资金外溢去补涨的次强个股，通常滞后龙头 1–2 个交易日启动，涨幅与高度低于龙头。补涨启动是板块扩散（赚钱效应外溢）的标志，但也意味着行情进入中后段。
  - **杂毛**：题材关联度低、流通盘小、纯靠情绪跟风的低位补涨股。涨时弹性大、跌时断得最快，是退潮期回撤幅度最大的群体。
- **板块高度的含义**：板块最高连板数即“高度”，它代表市场愿意为该方向接力的极限。高度抬升（如从 3 板到 5 板）说明情绪仍在升温；高度连续两日不创新高、龙头出现放量分歧或炸板，是高度见顶的关键信号。
- **联动的健康度判读**：
  1. 健康：龙头领涨开空间→中军跟上→补涨有序轮动，涨停梯队从高板到低板连续无断层。
  2. 转弱：梯队出现断层（如有 5 板和 2 板，中间 3、4 板空缺），说明接力链断裂，资金不愿在中间高度承接。
  3. 见顶：龙头炸板或中军集体走弱，补涨与杂毛却仍在拉升——这是资金在高位向低位“撤退式接力”，典型的退潮前夜。
- **梯队与仓位的对应原则**：主仓配置只给龙头与中军；补涨用小仓做扩散波段；杂毛原则上不碰，确需参与时仅用试错仓且当日不及预期即走。

### 3.4 卡位/占位个股的选择逻辑

“卡位”是在题材尚未形成绝对龙头、或龙头加速后资金寻找次优接力对象时，提前占据梯队中“逻辑正、位置好、有想象空间”的位置。卡位选股的核心是用结构换确定性。

- **卡位标的筛选条件（同时满足为佳，至少满足 4 条）**：
  1. **业务相关度高**：主营或在建项目与主线直接相关，营收占比可考证（见 3.6），而非互动易一句话蹭概念。
  2. **位置好**：股价处于相对低位或刚突破平台，上方无密集套牢区；优先选板块内“高度第二/第三”而非已经 5 板以上的龙头，给自己留接力空间。
  3. **流通盘适中**：游资主导的卡位常选流通市值 30–150 亿区间——太小易被一日游、太大拉不动。
  4. **筹码结构干净**：近一个月无巨量出货、无大股东减持公告、无解禁洪峰临近。
  5. **量能配合**：启动当日量比 ≥ 2、换手率在 5%–15% 的健康区间，封板有承接而非烂板。
- **卡位的两类时机**：
  - **主升前卡位**：板块刚被确认为主线、龙头尚未走出绝对空间时，卡位逻辑最正的中军或潜在龙二。这是赔率最高、也最考验对主线判断的时点。
  - **分歧转一致卡位**：龙头加速后高位分歧，资金回流寻找“补涨+逻辑正”的对象，此时卡位次强标的博一个补涨波段，止盈触发点要更紧。
- **买入触发点（示例，需结合个人系统）**：
  - 打板卡位：板块龙头封板且晋级率 ≥ 50% 的前提下，卡位股在 13:00 后放量过前高、封流比 ≥ 1% 时介入。
  - 低吸卡位：主线确立、卡位股缩量回踩 5 日线不破、分时回踩不创新低时分批介入。
- **卖出/止损触发点**：
  - 板块高度两日不创新高或龙头炸板，次日卡位股冲高不及预期（未能封板或封板后开板放量），减仓或清仓。
  - 个股跌破买入参考位（如买入分时均价或前一日实体阳线一半），无条件止损，单笔亏损控制在本金的可承受阈值内。

### 3.5 题材生命周期各阶段操作

一轮题材通常经历发酵—加速—高潮—分化—退潮五个阶段（部分阶段可能合并或跳过）。这一题材级序列与第二章的市场级六阶段（冰点→启动→发酵→高潮→分歧→退潮）大体对应：题材级“加速”≈市场级“发酵主升”，题材级“高潮”≈市场级“高潮”，题材级“分化”≈市场级“分歧”，题材级“退潮”≈市场级“退潮”——两者刻度不同（一个是单题材内部节奏、一个是全市场情绪温度），但攻防口径一致。核心纪律是：在大概率赚钱的发酵—加速段重仓，在大概率亏钱的退潮段空仓规避。

- **发酵期（情绪扩散、连板家数开始增多）**：
  - 信号：板块涨停家数从 2–3 家上升到 5 家以上，出现首个 2–3 连板，市场对该方向开始形成共识但尚有分歧。
  - 操作：这是上仓位最关键的窗口。重点卡位龙头/潜在龙二与中军，仓位可逐步加到计划上限。容错相对高，但要确认晋级率支撑。
- **加速期（主升、梯队完整向上）**：
  - 信号：龙头连续封板、高度持续抬升（如 4 板→6 板），中军放量上涨，补涨有序轮动，晋级率 ≥ 50%。
  - 操作：持有龙头与中军吃主升，补涨做扩散波段。不轻易在加速中段止盈，但要把跟踪止损上移到 5 日线或前一日封板分时低点。
- **高潮期（情绪亢奋、一致加速、龙头走出极限高度）**：
  - 信号：龙头走出 7 板以上甚至连续一字、市场情绪一致看多、连板家数与封板比创阶段新高、全市场注意力集中于此。
  - 操作：高潮往往就是顶部区。原则是“高潮兑现、不恋战”——龙头出现首次放量长上影、烂板、或集合竞价高开后快速跳水时，逐步止盈而非加仓。绝不在高潮期追买高位接力股。
- **分化期（梯队开始断层、强者恒强弱者出局）**：
  - 信号：龙头与跟风股走势分化，部分连板炸板、晋级率跌破 50%、梯队出现断层。
  - 操作：只保留最强龙头的剩余仓位并设紧止盈，清掉所有补涨与杂毛。分化期是“去伪存真”，不是抄底加仓的理由。
- **退潮期（情绪降温、普遍断板下杀）**：
  - 信号：龙头连续两日不封板或大面、连板家数断崖式下降、板块整体放量下跌、昨日涨停今日大面（“反包失败”普遍）。
  - 操作：空仓规避为主。退潮期补涨与杂毛回撤最深，严禁“跌下来便宜了就接”。等待该方向重新走出新的发酵信号（产业大趋势题材常有二波）再考虑参与。
- **阶段切换的客观警报（任一触发即降仓）**：板块高度两日未创新高；晋级率两日低于 50%；龙头首次炸板或放量长上影；当日板块涨停家数较前一日腰斩。

### 3.6 消息面与基本面纯正度评估

纯正度决定一只票能不能扛过分化、能不能在二波被重新选中。同样的主线里，逻辑纯正的标的更易获得机构与大游资认可、回撤后更易修复；蹭热点的伪标的退潮时回撤最深、易补跌甚至直接断头。

- **业务相关度核验（动手前必查）**：
  1. 查公司主营构成与最近一期财报，确认主线相关业务的**营收占比**：占比 ≥ 30% 视为高相关，10%–30% 为中等，< 10% 或仅“在建/规划/参股”属于弱相关。
  2. 查互动易/e 互动、公告原文，区分“已落地的业务/订单”与“尚在洽谈/拟布局/不构成承诺”的措辞——后者是典型蹭概念话术。
  3. 查项目是否有产能、客户、合同等可验证抓手，而非仅一条互动回复或一篇研报小作文。
- **概念真伪与“蹭热点”识别清单（命中越多越要回避）**：
  - 公告/互动内容与主营业务关联度极低（如纺织企业宣称做 AI 芯片、食品企业切入元宇宙）。
  - 表态时点与市场热点高度吻合，且用词模糊（“关注”“研究”“不排除”）。
  - 股价急涨急跌、脉冲后迅速回落，缺乏连续接力。
  - 伴随大股东/高管减持、定增解禁等利己动作。
  - 收到交易所**关注函/问询函/监管函**要求说明蹭热点情形——这是高危信号，常对应炒作终结。
- **纯正度分级与操作映射**：
  - 高纯正（高相关 + 有落地 + 无问询）：可做主仓卡位，享受二波修复。
  - 中纯正（中相关或落地存疑）：仅做主升段，分化期前清仓，不参与二波博弈。
  - 低纯正（弱相关 + 纯蹭 + 有问询函）：不纳入主仓，至多极小试错仓且当日兑现，退潮期坚决回避。
- **基本面红线（命中即排除出卡位池）**：流通盘存在临近大额解禁/减持、最新财报显示主业大幅恶化或预亏、涉及立案调查或退市风险警示（除非专门做 ST 博弈且明知其当前 ±5%、2026-07-06 起 ±10% 的限制与流动性风险）。

### 3.7 规避跟风杂毛与伪龙头

退潮期最大的亏损来自两类标的：跟风杂毛与伪龙头。前者靠情绪外溢生存、退潮即崩；后者顶着“龙头”名头却名不副实，是资金诱多的工具。

- **杂毛的识别与处置**：
  - 特征：题材关联度低、流通盘极小、无机构/大游资介入、纯靠情绪跟风补涨、龙虎榜净额小且席位杂。
  - 处置：原则上不纳入交易池；行情扩散期如确需博弈，仅用试错仓，且设定“当日封板失败/冲高回落即走”的硬触发点，绝不留隔夜过退潮。
- **伪龙头的识别（满足任意 2 条即高度警惕）**：
  1. **逻辑伪**：业务相关度 < 10%、纯蹭概念、收到问询函却仍被冠以“龙头”之名。
  2. **结构伪**：高度虚高但封板质量差（烂板、反复开板、封流比极低），没有中军与梯队呼应，是“孤独的高板”。
  3. **资金伪**：龙虎榜以游资对倒、买卖席位重叠、缺乏接力资金；或全靠一字板锁仓制造的“假强势”，一旦开板即巨量出逃。
  4. **位置伪**：股价已在高位、上方套牢盘厚重，所谓“龙头”实为高位诱多。
- **真伪龙头对照判别**：真龙头具备“高度最高 + 成交额前列 + 封板强 + 有中军梯队呼应 + 逻辑正 + 知名资金介入”六项共振；伪龙头往往只占其一二（如只有高度、没有厚度与逻辑）。卡位与打板只认真龙头，宁可错过不可错买。
- **退潮自保纪律**：一旦确认主线进入分化/退潮（参照 3.5 警报），优先清杂毛、再清补涨、最后处理龙头剩余仓位；任何时点都不以“跌得多、便宜”为由抄底杂毛与伪龙头。


## 四、龙头股辨识标准

龙头股是题材发酵周期里资金合力的集中体现，是连板梯队的最高点和情绪风向标。辨识龙头的本质，是判断"市场资金把哪一只票当成做多核心来反复套利"。龙头是市场用真金白银选出来的，不是盘前预判出来的——本章给出的所有标准都用于**事后快速确认与持续跟踪**，而非提前埋伏。下文阈值是过滤条件而非买入理由，任何单一指标都不构成买入依据，封单可撤、龙头可证伪，须多维交叉验证。

### 0. 制度前提（与本章阈值直接相关）

（交易制度统一口径见《制度与术语基线》一节，本章只列差异点。）辨识连板高度、封单强度、封板时间，必须先锁定标的所在板块的涨跌幅制度，否则"几个板""封单多大"没有可比性——同样的"连板数"在 ±20%/±30% 制度下空间与风险成倍放大，封单门槛、买入仓位需相应下调，北交所龙头辨识标准不可机械套用主板。本章涉及的差异点：

- **ST / \*ST**：主板风险警示股截至本手册撰写日（2026-06-13）仍执行 ±5%；沪深北交易所已发布修订规则，自 **2026-07-06 起主板风险警示股涨跌幅由 ±5% 调整为 ±10%**，创业板 / 科创板风险警示股为 ±20%。切换日前后以券商行情实际显示的涨跌停价为准。ST 股退市风险高，本章不将其纳入推荐龙头池。
- **次新妖股**："高度"不能用连板数衡量，须单独看换手与分时强度，不混入连板龙头标准。30% / 60% 两档盘中临时停牌（每次 10 分钟）只适用于"无涨跌幅限制"的股票（新股前 5 日、北交所新股首日），不是已上市股票的日常盘中机制。

### 1. 量化维度（硬指标，逐项打分）

#### 1.1 流通市值：适中区间

游资做盘的本质是"用可控资金撬动股价并能全身而退"，因此流通市值需落在**既能拉动、又能出货**的区间：

- **核心适中区间：流通市值约 30 亿–150 亿元**。这是连板龙头最高发的带；资金门槛低、筹码结构相对简单、"船小好调头"。
- **< 20 亿元**：易拉但流动性差，封单一旦松动无人接盘，出货困难，更易演变为庄股 / 杂毛行情，作为全市场龙头不优先。
- **150 亿–300 亿元**：可作"中军 / 权重型龙头"考量（见 §3.2），需要的承接资金量显著放大，连板高度通常受限。
- **> 300 亿元**：单日动用几十亿才推得动，基本退出纯情绪连板龙头的候选池。
- 辅助看**换手率**：龙头在**连板加速段日均换手常在 10%–30%（口径10②，连板加速段换手，与第七章首板换手口径10①区分）**；一字 / T 字板换手极低（高控盘），开板后换手骤升是分歧加剧信号。

#### 1.2 首次涨停时间与封板时间

涨停越早、越快、越稳，资金意图越强：

- **首板封板时间**：保守打板只参与 **10:00 前**封板的标的；**9:45 前秒板 / 一字板**为最强一档。10:00 后才首封，强度打折，临近尾盘（14:30 后）首封多为博弈盘，次日溢价不稳。
- **封板质量排序（由强到弱）**：一字板 > 秒板（开盘几分钟内） > 早盘换手板 > 反包板 > 烂板 / 尾盘偷袭板。
- **是否炸板**：当日开板次数越多越弱。**反复开板（≥2 次）**的"烂板"次日溢价大幅衰减，不计入强势龙头。
- **加速段位置**：龙头在 2→3 板、3→5 板的"换手加速板"最具号召力；连板后期（6 板以上）封板时间反而前移成一字，是高位锁筹，风险与空间同步放大。

#### 1.3 封单强度（封流比）

用**封流比 = 封单金额 ÷ 流通市值 × 100%** 量化封板质量，剔除流通盘大小干扰：

- **封流比 ≥ 2%**：封板较实的基准线；封流比越大，控盘越强、龙头地位越稳、行情持续性越好。
- **强势龙头**通常封流比在 **3%–10%** 以上；高位龙头一字板封流比可达两位数。
- **封流比 1%–2%**：封板强度一般；**封流比 < 1%** 或封单 / 流通市值明显偏薄：**弱封**，盘中撤单即开板，不可视为有效涨停。
- 配合看**封单绝对额**与**封单稳定性**：龙头封单应是单向加厚、午后不减；若封单震荡撤换、巨单频繁撤挂，是出货 / 试盘信号。注意大股东持股名义可流通实难抛售，必要时用"封成比"（封单 / 实际流通筹码）修正。

#### 1.4 人气与资金排序

- **人气榜 / 热度榜**：以东方财富股吧人气榜、同花顺热度榜 TOP 排名为参考，**全市场龙头应稳居人气榜前 3–5 名**，且排名上升而非掉落。排名持续居首但股价滞涨，是"人气透支"预警。
- **龙虎榜资金**：龙头加速段龙虎榜应见**知名游资席位 / 一线营业部买入**，且买一显著大于卖一、买方席位集中。出现**机构专用席位净卖出、游资席位对倒或集体撤离**，是龙头退潮的领先信号。需警惕"假机构 / 包装席位"，以净买额与席位连续性交叉验证，不被单日榜单造神误导。

### 2. 定性维度（逻辑指标）

#### 2.1 题材纯正度与唯一性 / 稀缺性

- **纯正度**：标的主营或核心逻辑必须**直接命中**当下发酵题材，而非"沾边蹭概念"。蹭概念票一旦被市场识别，溢价最先崩塌。
- **唯一性 / 稀缺性**：在该题材内具备**别人无法替代的标签**（独家产品、独家事件、唯一壳资源、唯一纯正概念股）。题材内可替代标的越多，越难走成总龙头。
- **逻辑级别**：政策级 / 产业级大题材（可支撑多梯队、跨周持续）优于事件级小题材；小级别"一周游"题材难出真龙头，补涨多失败（见 §3.3）。

#### 2.2 在板块中的位置

- **是否最高板**：真龙头通常是**板块连板高度的最高点**（空间板），是梯队的天花板。
- **是否带动板块**：龙头分时启动应能**带动板块跟风股集体异动 / 涨停**，即"龙头一拉，板块跟上"。只有自己涨、带不动同板块的，是孤儿行情，号召力存疑。
- **领涨顺序**：开盘 / 异动时**率先封板并触发板块效应**者，比尾随封板者地位高。

#### 2.3 空间高度与号召力

- **空间高度**：连板数代表市场对该逻辑的认可上限。高度本身不是越高越好，但"敢于打开空间、且打开后板块跟随"的标的，号召力最强。
- **号召力**：核心检验是——**它涨停，资金敢去板块里找补涨 / 跟风；它一旦炸板，整个板块跟随回落**。具备这种"涨跌都带节奏"能力的，才是真龙头。

### 3. 角色分类与区分

同一题材梯队内，不同角色的辨识标准与交易意义不同，须严格区分，避免把跟风 / 杂毛错当龙头追高：

#### 3.1 真龙头（总龙头 / 板块龙）

- 主要参与者为游资 + 散户，纯情绪投机，连板行情、市值适中、题材纯正且唯一。
- 满足：**最高板 + 早封强封（封流比 ≥ 2%）+ 人气榜前列 + 龙虎榜游资进场 + 能带动板块**。
- 是梯队的核心套利标的与情绪锚——**它不倒，板块不散；它倒，全队退潮**。

#### 3.2 龙头中军

- 板块内**市值偏大、有基本面 / 业绩支撑、机构参与度高**的趋势型品种，常是行业真实龙头（与连板情绪龙非同一只）。
- 判据是"**不倒**"：可以不连续涨停，但只要高位不大幅下杀，就稳住板块信心。中军走弱往往先于情绪龙见顶，是板块体温计。

#### 3.3 补涨龙

- **触发条件**：总龙头走到 **5 板及以上 / 退潮 2–3 天**，资金为维持热度向梯队下方挖掘。常从**首板或二板**启动补涨。
- **高度受限**：补涨龙高度被原龙头压制（老龙 7 板，补涨龙可能 5 板即止）。
- **题材门槛**：仅**大题材**能挖出有效补涨龙；小级别 / 杂毛题材补涨基本失败。补涨是退潮期博弈，安全边际低于主升期的真龙头。

#### 3.4 跟风 / 助攻 / 卡位

- **跟风**：龙头 1–4 板阶段、分时领涨带动板块涨停者，资金"赚个套利就跑"，多数二板兑现，**不具持续性**。
- **助攻**：龙头之后涨停的首板 / 二板，作用是抬升板块赚钱效应，非核心。
- **卡位**：日内卡位（先于龙头封板）一般 2 板结束；龙头卡位（龙二替代龙一）才有升级为新龙头的可能，须以"接棒后能否带动板块"重新验证。

#### 3.5 伪龙头 / 杂毛（排除项）

满足任一即判为伪龙头 / 杂毛，原则上不参与或最先止损：

- 蹭概念、题材不纯或逻辑被证伪。
- 弱封 / 反复炸板 / 烂板，封流比 < 1%。
- 流通市值过小（< 20 亿）的庄股式拉抬，或市值过大拖不动。
- 人气榜靠后且排名持续下滑、龙虎榜无主流资金 / 仅游资对倒。
- 自己涨停但带不动板块、无跟风效应的孤儿行情。
- 题材级别过小（一周游），全梯队仅 1–2 只孤票。

### 4. 筛选流程与排除项（可执行清单）

#### 4.1 "全市场最强一只" 筛选流程

1. **定题材主线**：复盘当日涨停 / 连板梯队，找出**连板家数最多、赚钱效应最强**的 1–2 条主线；无明显主线日，全市场龙头标准不成立，降低参与。
2. **定主线内最高板**：在主线内取**连板高度最高**的标的作为候选龙头（多只同高度时进入下一步比对）。
3. **量化复核**：逐项核对——流通市值 30 亿–150 亿、首封 10:00 前、封流比 ≥ 2% 且封单未减、人气榜前 3–5、龙虎榜游资 / 一线席位净买入。
4. **定性复核**：题材纯正且唯一、最高板、能带动板块、号召力（涨带板块、倒带板块）。
5. **跨主线择强**：若多条主线均有龙头，比较**连板高度、封流比、人气榜位次、龙虎榜资金体量**，取综合最强者为"全市场最强一只"。
6. **确认而非预判**：以上全部用于**封板后 / 次日竞价**的确认与持有跟踪，不用于盘前埋伏。

#### 4.2 "板块最强一只" 筛选流程

1. 锁定目标板块，列出当日板块内全部涨停 / 异动个股。
2. 取**最高板**为基准候选；同高度时比**首封时间 → 封流比 → 人气 / 龙虎榜资金**。
3. 验证**板块带动力**：该票分时启动时板块跟风股是否同步异动 / 涨停。
4. 区分**情绪龙（连板核心）**与**中军（市值 / 基本面核心）**，明确自己要参与的是哪一类，二者交易逻辑不同。

#### 4.3 统一排除项（命中即剔除候选龙头池）

- 题材不纯 / 蹭概念 / 逻辑被证伪。
- 首封晚于 10:00、当日反复炸板（≥2 次）、烂板。
- 封流比 < 1% 或封单盘中持续减少 / 巨单对倒撤换。
- 流通市值 < 20 亿（庄股式）或 > 300 亿（拖不动）。
- 人气榜排名靠后且下滑、龙虎榜无主流资金或仅席位对倒。
- 涨停但不能带动板块、无跟风梯队的孤儿行情。
- ST / \*ST、退市风险股，及高度无法用连板衡量的次新（须单独评估，不混入连板龙头标准）。
- 题材级别过小（预期"一周游"）、全梯队孤票。

#### 4.4 持续跟踪与证伪触发点

龙头辨识是动态过程，确认后须用以下信号判断龙头是否仍然成立（任一触发即降级 / 离场，具体卖出执行见相关章节）：

- 封流比骤降、盘中开板且回封无力。
- 次日竞价大幅低开 / 高开冲高回落，量能放大而不涨。
- 人气榜位次下滑、龙虎榜游资集体撤离或机构净卖出。
- 板块跟风股集体走弱、补涨龙启动（往往意味主升接近尾声）。
- 题材出现证伪利空或政策 / 监管降温。

> 本章特定风险提示：龙头身份由资金动态投票决定、可被快速证伪，"事后确认"意味着买点天然滞后于启动；务必以 4.4 的证伪触发点动态降级，不可因已确认龙头身份而放松止损。


## 五、集合竞价信息解读

集合竞价是每个交易日里"信息密度最高、噪声也最大"的几分钟。开盘竞价决定了打板/接力的起跑点与确定性，尾盘竞价则暴露主力对次日的真实预期。本章先厘清制度边界，再给出可量化的解读口径与决策矩阵。所有阈值均为经验区间，需结合个股流通盘与当日市场情绪动态校准，不存在固定不变的"标准答案"。

### 5.1 制度边界：三段竞价的撮合规则

（交易制度统一口径见《制度与术语基线》一节，本章只列差异点）。本节只强调与竞价解读直接相关的"可撤/不可撤"时点，这是识别真假申报的根基。

- **开盘集合竞价 9:15–9:25**，内部分两段：
  - **9:15–9:20（可申报、可撤单）**：系统接收申报也接受撤单。此阶段显示的"虚拟开盘参考价/匹配量"随时可被推翻，是虚假申报（俗称"钓鱼单""幌骗单"）的高发窗口。主力可在此挂巨量买单把虚拟价顶到涨停，制造抢筹假象，再在 9:19:30 前后集中撤单。**9:15–9:20 的竞价价格仅供参考，不可作为决策依据。**
  - **9:20–9:25（可申报、不可撤单）**：系统继续接收新申报，但**不接受撤单**（撤单指令无效）。此阶段挂出的单子真金白银、无法反悔，最能反映真实意图。**真正有效的竞价信息从 9:20:00 起才开始累积，重点观察 9:24:00–9:25:00 这一分钟。**
  - **9:25:00 一次性集中撮合**，按"成交量最大化 + 高于成交价的买单与低于成交价的卖单全部成交 + 多个价位满足时取最接近昨收的价格"三原则，产生当日**开盘价**。**9:25:00–9:30:00 这 5 分钟为盘前静默期，只接收申报、不撮合、不可撤单**，9:30:00 起进入连续竞价。

- **盘中临时停牌复牌**及部分板块的盘中集合竞价机制不在本章展开，逻辑与上同。

- **尾盘收盘集合竞价 14:57–15:00（沪深各板块含沪市主板均适用、全程不可撤单、不接受市价申报）**：14:57:00 起接收申报但不可撤单，15:00:00 一次性撮合产生**收盘价**。由于全程锁单，尾盘 3 分钟的量、价、剩余挂单方向，是主力对次日预期的"不可反悔表态"，解读价值高于盘中任意时段。

> 撤单口径速记：撤单只在开盘竞价 **9:15–9:20** 这一个窗口可行；连续竞价（9:30–11:30、13:00–14:57）的普通限价单可随时撤；**9:20 之后到收盘（含 9:20–9:25、9:25–9:30 静默期、14:57–15:00 收盘竞价）均不可撤**。

### 5.2 涨跌幅与新股制度对竞价解读的影响（差异点）

竞价的"高开几个点算多"高度依赖该股的涨跌幅上限，归类口径见《制度与术语基线》（主板 ±10%、创业板/科创板 ±20%、北交所 ±30%）。本章只点出与竞价解读直接相关的差异：

- **ST/\*ST**：主板风险警示股截至本手册撰写日（2026-06-13）仍执行 ±5%；沪深北交易所已发布修订规则，自 **2026-07-06 起主板风险警示股涨跌幅由 ±5% 调整为 ±10%**，创业板/科创板风险警示股为 ±20%。切换日前后以券商行情实际显示的涨跌停价为准。本战法原则上回避 ST 标的，其竞价"高开几个点"的尺度须按当时实际上限单独换算。
- **新股上市初期**：主板/创业板/科创板新股上市前 5 个交易日不设涨跌幅、第 6 日起恢复对应板块限制；北交所新股上市首日不设涨跌幅、次日起 ±30%。无涨跌幅约束期的个股（新股前 5 日、北交所新股首日）适用 30%/60% 两档盘中临时停牌（每次 10 分钟），不适用本章"高开几个点"的常规口径，应单独按弹性更大的尺度看待。
- **T+1**：当日买入次日方可卖出。竞价买入即锁定至少一个交易日的隔夜风险敞口——这是打板/接力一切博弈的根本约束。

**关键换算：高开百分比必须先除以该股涨跌幅上限，换算成"距离涨停还有多少空间"再判断。** 主板高开 5% 已用掉一半涨幅空间，而创业板高开 5% 仅用掉四分之一，二者含金量完全不同。下文凡给出百分比阈值，未特别说明者均以**主板 ±10%** 为基准；创业板/科创板请按上限比例放大约 2 倍区间。

### 5.3 高开/低开幅度区间的含义（以主板 ±10% 为基准）

竞价幅度反映的是**隔夜多空合力的强弱与方向**，但幅度本身有"过犹不及"的非线性特征：

- **低开 −3% 以下（深水低开）**：隔夜利空或主力出货意愿强，接力风险极高，原则上放弃；唯一例外是"利空已知 + 超跌 + 缩量低开企稳"的反包预期，需盘中确认，不在竞价阶段下注。
- **低开 0% ~ −3%**：情绪转弱或获利盘兑现，连板股出现这种竞价多为退潮信号，倾向减仓/放弃。
- **平开 −1% ~ +1%**：分歧明显，方向未定，连板股平开常意味着进入"分歧转一致"或"一致转分歧"的拐点，需结合量能判断，不宜竞价直接打。
- **小幅高开 +1% ~ +3%**：温和、健康的承接，是连板接力相对舒适的起点。资金愿意小幅溢价进场，留有上行空间，**是最适合做分歧转一致、低吸接力的区间**。
- **中幅高开 +3% ~ +5%**：强势，但已开始透支次日溢价空间。适合强势板块龙头，需要量能与封单同步配合，否则易冲高回落。
- **大幅高开 +5% ~ +7%**：超预期强势，但接力者**普遍套利空间被压缩到 3–5 个点**，且高位获利盘在竞价后集中兑现概率大；**距离涨停过近，反而是"被砸"的高发区**。
- **超大幅高开 +7% ~ +9.x%（贴近涨停）**：除非是确定性极强的一字/T 字排队，否则平开后到连续竞价阶段经常出现"高开低走、瞬间炸开"，**对接力者是陷阱区，原则上放弃竞价介入，改为盘中确认后再决策**。
- **竞价直接封板（+10%）**：见 5.6 一字板/T 字板专项。

经验法则可直接落地：

- **连板接力，最优竞价高开区间是 +2% ~ +5%**；低于此区间动能不足，高于此区间溢价透支且砸盘风险陡增。
- **"竞价大高开 + 竞昨比异常放大"要优先考虑被砸**（见 5.4）。这是最反直觉、也最容易亏钱的地方：高开越猛，未必越安全。

### 5.4 竞价量能：竞昨比、竞昨成额比与占流通比

只看价格会被诱导，**量价必须合看**。竞价量能用于判断"溢价是真金白银堆出来的，还是几手对敲画出来的"。以下指标以 9:25 撮合后的竞价成交量/成交额为准（小盘股自由流通市值 ≤30 亿元时区分度最好，大盘股需相应下调阈值）：

1. **竞昨比 = 竞价成交量 ÷ 昨日全天成交量**
   - 高开 + 竞昨比 **>10%**：放量，资金真实进场。
   - 若**昨日为一字板**（昨日换手极小），则该比值需 **≥200%** 才算真正放量（因为分母极小，必须大幅放大才有意义）。

2. **竞昨封单比 = 竞价成交量 ÷ 昨日分时最大单笔/峰值成交量**
   - 前提：昨日非一字板。该比值 **>50%** 才算爆量。

3. **竞昨成额比 = 竞价成交额 ÷ 昨日全天成交额**（按昨日成交额分档，避免大小票一刀切）：
   - 昨日成交额 <1 亿：竞价金额需 **≥15%**；
   - 1–2 亿：**≥10%**；
   - 2–5 亿：**≥15%**；
   - 5–10 亿：**≥8%–10%**；
   - 10–20 亿：**≥5%–8%**；
   - \>20 亿：**≥4%**。
   - 达到对应档位才算"竞价爆量"，资金承接才有说服力。

4. **竞价量占流通比 = 竞价成交量(手)×100 ÷ 流通股本(股)**——即"竞价换手率"。
   - 小盘股竞价换手 **>1%** 已属明显活跃，**>2%–3%** 属高度抢筹（也可能是高位放量出货，需结合价格方向）。
   - 该指标用于横向比较不同流通盘个股的资金集中度，**比绝对成交手数更可比**。

> 量价配合的核心结论：**温和高开（+2%~+5%）+ 竞昨比/成额比达标 = 真实抢筹（可接力）**；**大幅高开（>+7%）+ 竞昨比异常巨量（如竞昨比远超阈值、竞价换手 >3%）= 警惕高位放量诱多/对倒出货（倾向放弃）**。同样的"放量"，在不同高开幅度下含义相反。

### 5.5 抢筹/砸盘与撤单的诱多/诱空陷阱

竞价盘口的"动作"比静态数字更能暴露主力意图，重点看 9:19:30 与 9:24:00 之后的变化：

- **真抢筹特征**：9:20 之后（不可撤单段）才挂出的大买单、虚拟价稳步抬高、竞昨比持续放大、撮合前买盘厚度不减。这种"在锁单区加仓"的承接最可信。
- **诱多陷阱（最常见）**：9:15–9:20 把虚拟价顶到很高、匹配量很大，营造涨停在即的假象，吸引跟风挂单，随后在 **9:19:30 前后集中撤买单**，9:20 后真实买盘消失、开盘价大幅回落。**识别要点：9:20:00 前后虚拟价/匹配量出现断崖式下跌，即为撤单诱多。** 凡是高开主要由 9:20 前的可撤单堆出来的，一律按"假高开"处理。
- **诱空陷阱**：9:15–9:20 挂巨量卖单压低虚拟价制造恐慌，诱使持仓者在竞价/早盘割肉，9:20 后撤卖单、开盘反而走高。对持仓者而言，**不要在 9:20 前的可撤卖单压力下做决策**。
- **尾盘砸盘 vs 尾盘偷袭拉升**：
  - 尾盘 14:57–15:00 放量下杀、收盘剩大量卖单 → 主力出逃/规避隔夜风险，**次日低开概率大**，持仓者应在尾盘前减仓而非等竞价。
  - 尾盘几秒钟小量拉红、无持续买盘 → 多为"画 K 线"诱多，次日常高开低走，**不可因尾盘飘红而追接力**。
  - 尾盘放量上拉、收盘剩买单、且竞价换手达标 → 真抢筹，次日高开概率大，可纳入次日竞价接力观察池。

**一条铁律**：可撤单段（9:15–9:20）的任何"漂亮数字"都只是参考；**只认 9:20 之后不可撤的真实挂单与 9:25 撮合结果**。把决策权重几乎全部放在 9:24:00–9:25:00 这一分钟。

### 5.6 一字板、T 字板的竞价排队与换手竞价

涨停个股次日竞价直接封板时，分三种典型形态，确定性与可介入性递减/递增不同：

- **一字板（竞价即一字封停，全天几乎不开板）**
  - 买入只能在 9:15 开盘就以涨停价挂单**排队**，按"价格优先、时间优先"成交。能否买到取决于排在你前面的封单总量与当日开板放量。
  - 竞价封单越大、封流比（封单金额 ÷ 流通市值）越高，板越硬，但**散户成交概率越低**；竞价一旦出现封单逐步减少、9:24 后封单跳水，则有炸板风险。
  - **一字连板（连续一字、无换手）往往跑不赢有充分换手的接力标的**：因为没有换手意味着筹码未充分交换，一旦开板抛压集中、接力资金不敢进，反而容易闷杀。**一字连板的 1 进 2、补涨股 1 进 2、高位反包板，原则上不参与竞价排队接力。**

- **T 字板（竞价高开/涨停后盘中砸开、又封回，K 线留下长下影"T"形）**
  - 砸开瞬间是出货还是洗盘，关键看**砸开后的换手量与回封速度**：放量换手后快速回封（换手充分、封回坚决）相对健康；缩量砸开、回封无力则为出货嫌疑。
  - T 字板因为提供了买入位（砸开时可买到），接力可操作性高于纯一字板，但必须有量能与回封确认，不能在砸开下杀的过程中盲目低吸。

- **换手竞价（竞价不封板、留出换手空间）**
  - 竞价高开 +2%~+5%、竞昨比/成额比达标、买卖盘充分换手——这是游资接力**最偏好**的形态：筹码有交换、套牢盘被消化、有上行空间。
  - 换手竞价的"换手率达标 + 温和高开"组合，确定性通常优于"高封单一字板"，因为它体现的是**市场合力**而非单一资金锁仓。

封单的解读口径（统一用**封流比 = 封单金额 ÷ 流通市值**：≥2% 封板较实、1%–2% 一般、<1% 偏弱）：

- **封流比越大、且 9:20 后仍在增厚 → 越硬**；
- 9:24:00 之后封单**减少**、或撤封迹象明显 → 炸板预警，**降低接力优先级或放弃**；
- 一字板封单"看着大但散户排不进"≠ 可买，能否成交与板的硬度是两回事，二者要分开评估。

### 5.7 竞价排板未成交 / 高开未封时的处置（新增）

竞价排板与高开抢筹，最容易被忽视的是"挂单已发出、但开盘结果与预期不一致"的处置。由于撤单只在 9:15–9:20 有效（9:20 后即不可撤单，9:25–9:30 为只接收申报、不撮合的静默期），9:20 之后挂出的排板单一旦发出就无法撤回，必须在挂单前就想清楚成交与未成交两种结局的应对。

- **风险一：挂涨停价排板，可能在低于涨停的开盘价被动成交。** 集合竞价按"成交量最大化"原则在 9:25 一次性撮合，你以涨停价挂的买单，只要 9:25 撮合出的开盘价低于涨停，就会以**该（更低的）开盘价**成交，而不是只有封死涨停才成交。这意味着"想排一字板"却可能在高开未封的价位上拿到货，瞬间承担一个未封板标的的隔夜风险。
  - **应对**：只在**确认能封死涨停价**（封单远大于卖盘、虚拟价 9:24 后稳定顶在涨停且封单持续增厚）时才挂涨停价排板；若对能否一字封板没有把握，**不在竞价阶段挂单，改为 9:30 开盘后视盘口再决策**。

- **风险二：9:25–9:30 静默期无法撤单、也无法补单。** 9:25 撮合一出、开盘结果若不及预期（如高开未封、甚至高开转水），此时既不能撤掉已成交的仓位，也来不及在静默期内调整，只能等 9:30 连续竞价开始后处置。

- **9:30 后的处置：以分时第一笔承接量定去留。**
  - 开盘**未封涨停**但分时第一笔放量上攻、承接明显、迅速回封 → 视为强承接，可按原计划保留或补仓（仍受单票仓位上限约束）。
  - 开盘**高开低走、第一笔承接稀薄或直接向下破开盘价** → 判为高开诱多/承接不足，按预设止损位在 9:30 后**立即减仓或清仓**，不等"再看看"。
  - 凡是竞价被动成交、并非自己确认封板后主动买入的仓位，处置档位一律从严（按"未封板标的"而非"一字板"对待）。

### 5.8 打板接力/低吸的竞价决策矩阵

下表把"高开幅度 × 市场情绪 × 封单/量能"三维压缩为可执行口径，**输出仅为买入 / 减仓 / 放弃三类动作**，均以主板 ±10% 为基准（创业板/科创板按上限比例放大幅度区间约 2 倍）。表中"达标"指 5.4 的竞昨比/成额比/竞价换手满足对应档位。

#### 决策矩阵（接力买入视角）

| 竞价高开幅度 | 情绪高潮（赚钱效应强、连板高度延续） | 情绪分歧/混沌 | 情绪退潮（炸板多、跌停多、高度断板） |
|---|---|---|---|
| **低开 < −3%** | 放弃（除非已知利空+超跌，盘中再议） | 放弃 | 放弃 |
| **低开 −3% ~ 0%** | 减仓持仓 / 不新接 | 减仓 / 放弃 | 减仓清仓 / 放弃 |
| **平开 −1% ~ +1%** | 量达标且封单增厚→小仓试；否则观望 | 放弃 | 放弃 |
| **小高开 +2% ~ +5%** | 量价达标→**买入（首选区间）** | 量价达标→小仓买入，需盘中确认 | 放弃（退潮期高开易被反杀） |
| **中高开 +3% ~ +5% 偏上** | 龙头+封单硬+量达标→买入，仓位减半 | 观望/小仓 | 放弃 |
| **大高开 +5% ~ +7%** | 仅龙头一线、封单极硬可小仓博弈，否则放弃 | 放弃 | 放弃 |
| **超大高开 +7% ~ +9.x%** | 原则放弃，改盘中确认；竞昨比异常→警惕被砸 | 放弃 | 放弃 |
| **竞价封板（一字/T 字）** | T 字放量回封→可接；一字连板原则放弃排队 | 放弃 | 放弃 |

矩阵使用要点：

- **情绪是一票否决项**：无论高开多漂亮，**退潮期一律下调一档处理或直接放弃**。高度断板、跌停家数明显放大、昨日打板今日大面，都是退潮信号。
- **量能是确认项**：高开幅度合适但竞昨比/成额比/竞价换手**不达标**，一律降级为"观望/放弃"——没有真实资金的高开是无根之木。
- **封单是硬度项**：决定能否买到、板是否会炸；9:20 后封单减少即下调优先级。
- **大高开 + 巨量** 与 **温和高开 + 适量** 的处置相反：前者偏向放弃（透支+砸盘风险），后者偏向买入（健康承接）。这是全章最需要内化的反直觉点。

#### 持仓者减仓/止盈视角（昨日已打板，今日竞价该不该走）

- **次日竞价高开 +5% 以上但量价背离（竞昨比畸高、价格冲高回落）**：兑现部分或全部利润，不恋战。
- **次日竞价高开 +2%~+5%、量价健康、封单增厚**：可持有看盘中，或部分止盈降风险。
- **次日竞价平开/低开 + 缩量**：分歧加剧，**减仓优先**；若叠加退潮信号则清仓。
- **次日竞价深水低开 −3% 以下**：承认打板失败，竞价或开盘即止损，不赌反包。
- 任何情形下，**T+1 锁定意味着竞价是当日第一个、也是相对充分流动性的离场机会**，不要把希望全寄托在盘中。

### 5.9 竞价阶段操作检查清单

竞价时间极短，建议把判断流程固化为清单，按 9:20、9:24、9:25 三个时点逐项核对：

- **9:15–9:20（只看、不信）**
  - [ ] 记录虚拟开盘价、匹配量，但**不据此下单**；
  - [ ] 标记是否存在"可撤大单顶价"的诱多迹象，留待 9:20 后验证。
- **9:20:00（切换到真实信息）**
  - [ ] 虚拟价相对 9:19:30 是否断崖回落？（是→撤单诱多，"假高开"）
  - [ ] 真实买盘/卖盘厚度、方向。
- **9:24:00–9:25:00（决策窗口）**
  - [ ] 高开幅度落在哪一档？换算成"距涨停空间"是否合理？
  - [ ] 竞昨比 / 竞昨成额比 / 竞价换手是否达标（按 5.4 分档）？
  - [ ] 形态属一字 / T 字 / 换手竞价中的哪一类？封单是否在增厚？
  - [ ] 若挂涨停价排板，是否已确认能封死涨停？（不能确认→不挂，改 9:30 后决策，见 5.7）
  - [ ] 当日市场情绪处于高潮 / 分歧 / 退潮哪个阶段？
  - [ ] 查矩阵 → 输出买入 / 减仓 / 放弃，并预设**止损位与最大隔夜仓位**（单票仓位上限：10cm ≤20%、20cm ≤10%、30cm ≤7%）。
- **9:25:00 撮合后（含 9:25–9:30 静默期）**
  - [ ] 实际开盘价与竞价预期是否一致？不一致（尤其大幅回落）→ 降级或放弃；
  - [ ] 静默期内无法撤单，9:30 后以分时第一笔承接量定去留（见 5.7），**不为了"已经看好"而强行执行原计划**。

### 5.10 本章特定风险提示

竞价阶段有一个其它章节不覆盖的特定风险须单独内化：**9:25–9:30 静默期内，已成交仓位无法撤单、无法调整**，因此竞价排板/高开抢筹的所有止损与减仓预案，都必须在 9:25 撮合之前就想清楚，并在 9:30 连续竞价一开始即机械执行，不能指望静默期内补救。


## 六、K线形态与分时盘口

打板与接力本质上是用次日溢价博弈承接资金的连续性，胜负在很大程度上取决于"形态有没有给出加速的理由"与"盘口有没有给出主力真买的证据"。本章把判断拆成两层：日线级别先用 K 线形态筛掉绝大多数不该碰的标的，盘中再用分时与封单逐分钟校验封板意愿与真实买卖点。所有阈值均为经验区间，须结合当时的市场温度（连板高度、炸板率、亏钱效应）动态收紧或放宽，弱势环境一律取严档。

> （交易制度统一口径见《制度与术语基线》一节，本章只列差异点）与本章直接相关的差异点是 ST/\*ST 制度切换：主板风险警示股截至本手册撰写日（2026-06-13）仍执行 ±5%，但沪深北交易所已发布修订规则、自 **2026-07-06 起主板风险警示股涨跌幅由 ±5% 调整为 ±10%**（创业板/科创板风险警示股为 ±20%），切换日前后以券商行情实际显示的涨跌停价为准。ST 标的封板含金量、连板节奏会因板幅放大而被重估，本战法原则上回避 ST 标的。

### （一）利于打板/接力的 K 线形态

下列形态的共同逻辑是：筹码经过充分换手或洗盘后趋于一致，且当日存在"突破/加速"的技术理由，从而提高次日资金接力的概率。

1. **首板平台突破**
   - 定义与条件：股价在一个相对横盘的箱体内运行 ≥10 个交易日，箱体高度（最高/最低）控制在 15% 以内；突破当日以涨停（或放量长阳）有效升破箱体上沿。
   - 量能要求：突破日成交量较平台期日均量放大 1.5–3 倍，首板健康换手参考 25%–40%（<10%~15% 偏低、缩量一字除外，>50% 才算筹码过度松动；流通盘越大取下限）。
   - 买点：竞价或盘中升破箱体上沿且不回落即为试仓点；最稳的是涨停封板那一刻打板。
   - 失效线：次日跌回箱体内部（跌破突破日实体下沿），视为假突破，无条件离场。

2. **横盘蓄势放量长阳**
   - 定义与条件：前期一波上涨后高位横盘 5–10 日，K 线缩量、重心不下移；某日放量收出实体 ≥7% 的长阳或直接涨停，重新打开上行空间。
   - 量价配合：横盘期量能逐日萎缩至高峰量的 30%–50%，启动日量能突然放大，呈"地量见地价、放量再启动"结构。
   - 买点：放量长阳确认升破横盘上沿后介入，涨停封板为最强信号。

3. **反包阳（含弱转强反包板）**
   - 定义与条件：前一日为阴线（最好是涨停后回调阴线，未跌破关键支撑），次日以阳线吞没前一阴线、收盘价 ≥ 前一阴线最高价（完全反包），力度上要求快速、最好上午封板。
   - 强弱判断：上午封板 > 下午封板；放量一字/秒板式反包 > 拉锯反包；阳线实体完全吞没前阴 > 仅部分吞没。
   - 买点：盘中放量升过前一日阴线最高点且站稳，为反包确立点；封板可打。
   - 注意：反包是强势股第二波机会，但前一阴线若已跌破 5 日线或放出天量，反包不成立，按规避形态处理。

4. **中继形态（缩量调整后再涨停）**
   - 定义与条件：连板或主升途中出现 1–3 日缩量调整（阴线但不破 5 日线、调整幅度小于前一根阳线实体的一半），随后放量再封板，确认调整结束、趋势延续。
   - 关键指标：调整期量能萎缩、换手温和（连板加速段龙头日均换手参考 10%–30%，锁仓中换手下降）；再启动日量能温和放大而非爆量。

5. **回踩不破均线**
   - 定义与条件：上升趋势中股价回踩 5 日线或 10 日线获支撑，下影线收回、收盘重新站上均线；均线系统保持多头排列（短期均线在上、依次向上发散）。
   - 买点：回踩不破均线 + 当日重新放量上攻时介入；跌破并收于关键均线之下则形态破坏。
   - 提示：若均线未形成多头排列、不处于突破/加速阶段，涨停的持续性显著下降。

6. **二波启动（龙回头）**
   - 定义与条件：龙头第一波连板后断板，经过 3–7 日横盘整理（阳多阴少、重心不下移、守住关键均线/平台），再次放量突破整理平台启动第二波。
   - 介入纪律：横盘期"不破关键支撑线"为底线，破位即放弃；二波启动点要求放量突破前期整理平台上沿，最好以涨停确认。

> 形态使用顺序：先确认大盘与板块温度→再用上述形态做候选池筛选→最后由分时盘口决定是否扣动扳机。任意一层不达标即放弃，不在弱势环境硬打。

### （二）必须规避的 K 线形态

下列形态的共同逻辑是：上方套牌沉重、获利盘集中兑现或趋势已破坏，次日承接资金断裂概率高，打板/接力的风险收益比严重恶化。

1. **高位放量长上影**
   - 特征：相对高位放出大量，盘中冲高后被砸回，留下长上影（上影长度 ≥ 实体，且上影 ≥ 当日振幅的 1/3），收盘远离最高价。
   - 含义：上方抛压凶猛、主力出货嫌疑大；高位 + 天量 + 长上影是典型见顶组合，严禁追板。

2. **断头铡刀**
   - 特征：一根大阴线（实体 ≥7%）一次性向下击穿 5 日、10 日、20 日（乃至 60 日）多条均线，且放量。
   - 含义：多头结构被一刀切断，短期趋势走坏，反弹只用于减仓不用于接力。

3. **跌破关键均线**
   - 特征：收盘有效跌破并站不回 5 日线/10 日线/趋势平台颈线（升破后又被打回）。
   - 处置：打板标的一旦次日跌破买入日实体下沿或关键均线，按止损纪律无条件离场；不要用"再等等"来对抗趋势破坏。

4. **高位巨量阴线**
   - 特征：连续上涨/连板后，高位放出超过前期峰值量的巨量，收中/大阴线，换手率异常放大（已突破筹码过度松动线，常 >50%）。
   - 含义：分歧转一致向下、主力借量出货，是连板行情终结的高频信号，次日大概率低开杀跌。

5. **双头（M 头）**
   - 特征：股价两次冲击相近高位均未能有效突破，形成两个对称头部，第二个头部往往缩量，跌破颈线（两头之间的低点连线）确认形态。
   - 含义：上方阻力被反复验证，趋势由上转下，跌破颈线后量度跌幅指向更低位置，禁止在颈线附近接力。

**规避形态速查清单（命中任意一项即不打）**
- [ ] 高位 + 天量 + 长上影/大阴线（出货三联）
- [ ] 一阴破多线的断头铡刀
- [ ] 收盘有效跌破 5/10 日线或趋势颈线且站不回
- [ ] 连板后换手异常放大（>50% 筹码过度松动）的巨量阴线
- [ ] 双头/双顶跌破颈线
- [ ] 形态处于明显高位且无新增上行空间（突破/加速理由缺失）

### （三）分时与盘口：封板质量分级

打板的真正确认发生在涨停那一刻及之后的盘口表现。封板质量决定次日溢价，核心看三件事：**封板速度、封单稳定性、开板后的再封能力**。

1. **按封板时间与速度分级**
   - 秒板/一字板：集合竞价即封死或开盘数秒封板，封单巨大、全天不开。资金一致性最强，次日溢价预期最高，但往往买不进（需竞价抢或排板）。
   - 快板（早盘强势板）：上午 9:30–10:30 一气呵成放量封板、封后不松动。质量优良，是打板主力时段。
   - 慢板（午后/震荡封板）：盘中反复上攻、临近午后或下午才封板。封板意愿一般，需结合封单稳定性二次确认。
   - 弱化分级口径：同等条件下"早封 > 晚封、快封 > 慢封、放量秒封后无开板 > 拉锯式封板"。

2. **封单强度的量化参照（统一用封流比，须随市场温度调整）**
   - 封单强度统一用"封流比 = 封单金额 ÷ 流通市值"：**≥2% 封板较实、1%–2% 一般、<1% 偏弱**。封流比过小易被砸开，封流比看着很大也可能是对倒虚封，须结合分时验证。
   - 封单量/当日成交量：该比值越大、封板越牢。经验区间——
     - ≥10:1：封板牢固，次日开板概率低；
     - 5:1–10:1：相对稳固，盘中可能短暂打开但回封概率高；
     - <5:1：封单偏薄，次日开板/低开风险明显上升。
   - 封单动态：观察"封减/封增"。封单持续被吃、逐级减少（封减）是松动前兆；封单逐步增厚（封增）是意愿增强。临近尾盘封单大幅抽离，警惕次日低开。

3. **烂板与反复开板**
   - 定义：涨停后多次被打开又回封，全天反复炸板，最终勉强收涨停。
   - 含义：多空在涨停价剧烈分歧、主力实力或意愿不足；烂板次日溢价差、低开概率高，原则上不打、已持有者优先在次日竞价高开时减仓。
   - 例外提示：极强主线龙头的"换手烂板"（巨量换手洗盘后仍封住）与垃圾股的"无量烂板"性质不同，须结合板块地位区分，但对追板者整体仍属高风险，宁可错过。

4. **炸板（放量打开）后的再封能力——这是核心买卖点判据**
   - 打开后能否快速、放量再封死，是判断主力封板意愿是否坚决的最直接证据：
     - 打开后 5–10 分钟内放量迅速回封、且回封后封单显著增厚→意愿坚决，可视为"洗盘式炸板"，回封点可作为打板介入点（弱市仍需谨慎）。
     - 打开后量能涌出、迟迟封不回，或封回但封单单薄、反复掉封→意愿存疑，视为出货嫌疑，不接力、持有者借反抽减仓。
   - 量化触发：炸板回封点的成交量应明显大于打开时的卖出量，呈"打开放量—回封再放量—封后缩量"结构；若回封无量或封后封单不及打开前的一半，判为弱回封。

### （四）分时逐笔与盘口语言：识别真买与衰减

分时不仅看价格曲线，更要读"逐笔成交与盘口挂单"这套主力语言，用以区分真买真卖与做盘诱导，并据此给出减仓判据（与第十/十一章止损口径打通）。

1. **对敲大单识别**：分时出现连续等量的大买大卖（买卖单笔数、手数高度对称、价位贴着同一区间反复成交），而价格几乎不动、成交量却不断堆积，多为主力对倒虚增量能、制造活跃假象，不能当作真实承接，需结合封流比与封单稳定性证伪。
2. **内外盘比口径**：外盘（主动买、以卖价成交）持续显著大于内盘（主动卖、以买价成交）反映买方主动性强；若量能放大但内盘反超外盘、且价格滞涨，是借量出货嫌疑。内外盘只看比例方向、不看绝对数，须与对敲识别交叉验证以剔除虚假活跃。
3. **多波拉升的衰减判据**：分时呈数波拉升时，逐波比较"量能与拉升角度"——后一波量能小于前一波、拉升角度变缓（爬升变钝、上攻乏力），即视为承接力衰减，按预案减仓，不等明确破位；衰减出现在涨停板被打开或封单同步走薄时尤须果断。

### （五）特殊分时结构与买卖点判定

1. **分时回踩不破**
   - 形态：分时图冲高后回踩分时均价线（黄线）或当日重要支撑（如开盘平台、前一日涨停价），不破并重新上攻。
   - 应用：分时回踩均价线获支撑、缩量企稳后再放量拉升，是日内相对安全的低吸/半路介入点；有效跌破分时均价线且放量下行，则放弃当日操作。

2. **尾盘偷袭拉板**
   - 形态：14:30 之后、尤其尾盘 15 分钟突然放量快速拉升封板。
   - 判别要点：
     - 区分"有量急拉"与"无量偷袭"：有量急拉常为主力对敲做盘，需警惕次日不接力；真正有效的尾盘板多见于前期底部已放量、有一定涨幅、沉寂后尾盘放量快速封死的标的。
     - 拉升角度越陡、封板越早越干脆、封后封单稳定，次日高开高走概率相对越高；尾盘最后几分钟才勉强封、封单薄的偷袭板，次日兑现/低开风险大。
   - 纪律：尾盘板留给次日的验证时间极短（T+1 无法盘中纠错），仓位务必从严，且优先打"封单稳、有量、形态在突破位"的尾盘板。

3. **天地板 / 地天板**
   - 天地板：当日从涨停（或高位）一路杀至跌停，主板振幅可达约 20%、创业板/科创板更大。多见于中小盘、次新、ST/\*ST 等，是资金出逃/情绪崩塌的极端信号——当日及次日严禁抄底接力，已持有者反弹即走。
   - 地天板：从跌停拉升至涨停，振幅同样极端。可能是恐慌错杀后的资金反抢，也可能是诱多。介入前必须用封单稳定性与回封能力二次确认（参照本节封单与再封判据），不可仅因"反转幅度大"就追入。
   - 共性：天地板/地天板都意味着当日多空分歧达到极致、波动率失控，对 T+1 的追板者属高赔率但更高风险结构，仓位与止损必须前置设定。

### （六）用分时盘口判断主力封板意愿与真实买卖点：综合检查清单

下表把日线形态与分时盘口合并为一张"扣扳机前必过"的清单，任一硬性项不达标即放弃当笔交易。

1. **形态层（日线，候选池准入）**
   - [ ] 处于突破/加速阶段，均线多头排列，非明显高位；
   - [ ] 命中利于打板的具体形态（平台突破/横盘放量长阳/反包/中继/回踩不破均线/二波启动）之一；
   - [ ] 未命中任何规避形态（长上影/断头铡刀/破关键均线/高位巨量阴线/双头）。

2. **盘口层（分时，扣扳机确认）**
   - [ ] 封板时间偏早（优先早盘快板/秒板），非午后勉强封；
   - [ ] 封单量/成交量 ≥5:1（追求稳健取 ≥10:1），封流比 ≥2%（≥2% 封板较实、1%–2% 一般、<1% 偏弱）且呈封增；
   - [ ] 若曾炸板，已在 5–10 分钟内放量回封且封单增厚（洗盘式而非出货式）；
   - [ ] 逐笔无明显对敲虚封、外盘主动性强于内盘、多波拉升未出现量能/角度衰减；
   - [ ] 分时未跌破均价线、未出现尾盘封单大幅抽离；
   - [ ] 非烂板、非天地板当日、非无量虚封。

3. **买点触发（满足形态层+盘口层后）**
   - 打板点：涨停封板瞬间排板/扫板买入（一字板用竞价抢）；
   - 半路点：分时回踩均价线不破、放量再上攻时介入；
   - 接力点：次日竞价对前日强势板做承接，须结合次日竞价量与高开幅度（弱转强、缩量上板优先）。

4. **卖点 / 止损触发（T+1 不可盘中纠错，须机械执行）**
   - 次日竞价：低开幅度超过预设阈值（如低开 >3%–5%）或竞价持续走低，开盘即减/清；
   - 盘中破位：跌破买入日实体下沿或关键均线（5/10 日线）无条件止损；
   - 封板恶化：盘中封单被持续吃穿（封减加速）、炸板后无量回封或封不回，反抽即出；
   - 分时衰减：多波拉升后一波量能/角度衰减、或逐笔转为内盘主动放量，即按预案减仓不恋战；
   - 高位见顶信号出现：当日走出长上影/高位巨量阴线/断头铡刀，次日不恋战。

> 本章特定提示：对敲虚封、尾盘诱多、地天板二次诱多都属本章特有的盘口陷阱，逐笔与封流比证据互为印证、缺一不可，任一项证伪即放弃当笔，不靠单一指标下注。


## 七、量能与换手分析

价格是结果，量能是过程。短线游资战法的核心矛盾，是判断"承接资金愿不愿意以更高的价格接走筹码"，而量能与换手率正是这一意愿最直接的量化表达。本章不讨论趋势方向，只讨论同一根 K 线、同一段连板序列内，**用量能去回答三个问题：能不能打、能不能拿、是不是该走**。所有阈值均为经验区间，须结合板块属性、流通盘大小、市场情绪周期共同使用，不构成机械信号。

> （交易制度统一口径见《制度与术语基线》一节，本章只列差异点。）与量能解读直接相关的差异点：
> - 主板普通股涨跌幅 ±10%（ST/\*ST 按基线口径执行，本战法原则上回避 ST 标的）；连板高度、封单、换手须先按标的所在板块的涨跌幅口径归一后再横向比较。
> - 注册制下次新股（主板/创业板/科创板上市前 5 日、北交所新股首日无涨跌幅约束）流通盘小、参与活跃，换手率天然偏高，下文换手阈值需相应上浮。
> - T+1 下，"封单稳固度"决定的是**次日溢价兑现能力**，而非当日止损能力。
> - 竞价量能与封单的真伪判断高度依赖两段集合竞价：开盘集合竞价 9:15–9:25（9:15–9:20 可撤单、9:20–9:25 不可撤单，9:25 撮合出开盘价），收盘集合竞价 14:57–15:00（全程不可撤单）。

### （一）量比：盘中即时强度的标尺

量比 = 现成交总手 / （过去 5 日每分钟平均成交手数 × 当日已开盘分钟数），衡量当下成交密度相对近 5 日的倍数，**1 为分水岭**。它最适合用于盘中（尤其早盘）即时强弱判断，而非日线级别。

- **< 0.8**：明显缩量，关注度低、分歧小，多维持原有趋势，不宜作为打板理由。
- **0.8–1.5**：温和，量能与近期持平，趋势延续概率高。
- **1.5–2.5**：温和放量，是健康的攻击量。若股价同步缓升，多方占优，可继续持有/介入。
- **2.5–5**：明显放量，常对应突破、异动或分歧加剧，需结合价格方向判断是攻击还是出货。
- **> 5（尤其 > 8）**：异常放量。早盘出现可能是抢筹，但若出现在**高位或尾盘**，警惕主力对倒派发。这种量比通常无法全天维持，纯粹高量比不构成持有理由。

**早盘量比的实战口径**：竞价及前 15 分钟量比能否站上前一日的对应水平，是判断"今日是否有增量资金"的第一道闸。打板标的，9:25 竞价量 / 昨日全天量 < 1% 且无明显买盘堆积，多为弱势承接，谨慎追。

### （二）换手率：按流通盘分档的健康区间

换手率 =（成交股数 / 流通股本）× 100%。**同一换手率对不同流通盘含义完全不同**，必须分档看，不能用一个固定数字套所有票。下表为日换手率的经验分档（普通趋势股口径，非涨停日）：

| 流通盘 | 偏冷（缩量） | 健康活跃 | 过热预警 | 见顶/出货高危 |
|---|---|---|---|---|
| 小盘（< 30 亿） | < 3% | 5%–15% | 15%–25% | > 25%（高位） |
| 中盘（30–100 亿） | < 2% | 3%–10% | 10%–20% | > 20%（高位） |
| 大盘（> 100 亿） | < 1% | 2%–8% | 8%–15% | > 15%（高位） |

使用要点：

- **位置决定性质**：同样 20% 的换手，发生在低位首次启动是"换手充分、筹码交换、利于续涨"；发生在连板高位则是"分歧放大、主力派发"。**脱离股价位置谈换手率没有意义**。
- **次新股、注册制板块（创/科/北）阈值整体上浮**：流通盘小、参与者活跃，10%–30% 换手属常态，不可照搬主板小盘"超 15% 即过热"的标准。
- **连续地量横盘后首次换手放大**（如小盘股从 < 2% 跳升至 8%–12%）且价稳，是资金进场信号，优先级高于单纯看价格。

### （三）放量 / 缩量的语义与六种基础量价组合

放量、缩量永远是**相对**概念——相对前一日、相对近 5 日均量、相对自身历史。最常用的对照是"较前一日"：当日量 > 昨日量为放量，显著低于为缩量。六种基础量价组合的短线含义：

1. **量增价涨**：最健康的攻击形态，多方主导，可跟随。
2. **量增价平 / 价滞**：高位是派发预警（放量滞涨），低位可能是吸筹。
3. **量增价跌**：高位为出货，下跌初期为恐慌，需结合位置。
4. **量缩价涨（缩量上涨）**：抛压枯竭、惜售，常见于强势中继或一字加速，**但需警惕"无量空涨"后的补跌**。
5. **量缩价平**：观望，趋势延续。
6. **量缩价跌（缩量阴跌）**：杀跌动能不足，常见于中继整理末端，不宜恐慌割肉。

### （四）量价配合与量价背离

- **量价配合（同步创新高）**：价格创阶段新高时成交量同步放大，说明买盘介入力度持续加强，趋势可信，持有逻辑成立。
- **量价背离（顶背离）**：价格创新高但成交量**未能同步创新高甚至萎缩**，是买盘力度衰竭的标志，涨势可疑，应将止盈/减仓阈值收紧。
- **底背离**：价格创新低但成交量持续萎缩、缩量不再扩大，杀跌动能耗尽，是企稳的前提（非买入信号本身）。

实战判断口径：连板途中，**第 N 板的攻击量明显小于第 N-1 板、而股价反而要更高位强行拉板**，即典型量价背离，是炸板/高开低走的高发结构，应主动降低仓位预期。

### （五）地量地价、天量见顶与缩量中继

- **地量地价**：股价低位 + 成交量创阶段地量（如小盘股换手 < 1%–2%、量能跌破近期均量一半）。表示抛压衰竭、无人愿卖，是底部区域的**必要非充分条件**。操作上等"地量后首次温和放量 + 价格站回关键均线"再动手，不抄地量本身。
- **天量见顶**：高位放出历史级巨量（远超前期任何单日，常伴换手率冲到上表"高危"区间），多为主力借对倒制造活跃假象、大规模派发。**天量后无法再创新高，且次日缩量回落，确认阶段顶**。短线纪律：高位见天量当日不追，持仓者优先在尾盘或次日冲高减仓。
- **缩量中继**：上涨趋势中途出现缩量小幅回调或横盘整理，量能萎缩说明**惜售、抛压轻**，是健康的洗盘而非出货，常孕育下一波。区分缩量中继与缩量下跌出货的关键：中继发生在均线多头、回调不破关键支撑（如 5/10 日线），且缩量幅度大（量能掉到前高的 40%–60% 以下）。

### （六）连板过程中的换手、封单与"放量板 / 缩量板"

连板序列里，量能的核心矛盾是**筹码松紧**：换手是为了"换手"——让低成本获利盘兑现、高成本接力盘进场锁仓，从而为次日续拉减轻抛压。

#### 放量换手板 vs 缩量一字板

- **缩量一字板**：竞价一字涨停封死、全天无成交或换手极低（常 < 2%–3%）。**封得最死、最强势，但买不进**——只能竞价排队挂单博概率成交，散户极难拿到货。其风险在于：一旦某日打开，因前期完全没换手、获利盘全堆在板上，**砸盘动能极大，易直接闷杀**。所以一字板"强"在封板、"险"在开板。
- **放量换手板（换手板）**：当日放量、换手充分（首板理想区间见下）封住涨停。**强在筹码经过充分交换、接力盘进场、续板更扎实**；弱在当日有真实抛压被消化，封单通常不如一字板厚。游资接力点多在换手板而非一字板。

#### 首板换手率参考区间

- **首板理想换手区间约 25%–40%（小盘/次新可更高）**：换手过低（< 10%–15%，缩量一字除外）说明接力意愿不足、关注度低，二板概率打折；换手 **> 50% 才算筹码过度松动**，获利盘随时反水，次日易高开低走。该区间与第九章首板筛选口径一致。
- **连板加速段另有口径，不可与首板混用**：连板加速段龙头日均换手约 **10%–30%**，且随锁仓加深换手反而下降；加速段换手骤升、甚至冲入上表"高危"档，往往是分歧加剧/主力派发，而非首板那种"换手充分利于续涨"。
- 一字首板因无法换手，须改看**封单厚度与竞价承接**，不能用换手率衡量。

#### 封单量与封板稳固度

封单量须结合流通盘看，**绝对金额无意义，比例才有意义**。两个常用比值（本节为封流比的权威定义出处之一）：

1. **封流比 = 封单金额 ÷ 流通市值**：衡量封板相对该票体量是否够厚，是全手册统一的封单强度口径。经验分档：**≥ 2% 封板较实、1%–2% 一般、< 1% 偏弱**（小盘股更易做到高封流比；高位龙头一字板封流比可达两位数）。
2. **封成比 = 收盘封单量 / 当日成交量**：衡量封单相对当日真实换手的厚度，是判断**次日溢价**的常用经验对照（仅作参考，非精确预测）：
   - 封成比 > 10：次日高开（或一字续板）概率高，多在 +7% 以上；
   - 3–10：次日多高开 6%–10%；
   - 1–3：高开 3%–6%；
   - 0.5–1：高开 1%–3%；
   - < 0.5：次日多平开甚至低开，封板脆弱。

封单**动态变化比静态值更重要**：尾盘封单持续增厚 = 看涨一致、次日续板概率高；封单大幅缩水、反复撤单堆单（对倒撤封）= 主力出货嫌疑，开板/炸板风险骤升。开盘集合竞价中 9:15–9:20 可撤单段的封单尤其要打折看真伪——9:20 后（不可撤单段）仍在的封单才相对可信。

### （七）分歧日放量与一致日缩量（情绪周期的量能特征）

连板高度题材的演绎，遵循"分歧—一致"循环，量能特征恰好相反，是判断接力节奏的关键：

- **分歧日放量**：龙头出现大幅低开、震荡、炸板回封，当日**天量换手**（如高位换手冲到 30%–50%+）。这是多空激烈交换、获利盘与接力盘对赌的标志。**分歧转一致**（低开高走、放量回封）是超短最经典的低吸/打板买点；**分歧加剧无法回封**则是退潮起点。
- **一致日缩量**：次日所有人看法趋同，一字板或秒板封死、**换手极低**。表面最强，实则**风险在积累**——因为没有换手意味着筹码没交换、获利盘越堆越高，一旦次日转分歧，砸盘动能巨大。**连续一致缩量后的第一个分歧放量日，往往是高度见顶或大幅波动日**。

实战节奏：偏好在"分歧日放量回封"出手，对"连续一致缩量"的高位板保持警惕——它给的是确定性溢价，但兑现窗口极短，且见顶常发生在最一致、最缩量的那一天之后。

### （八）用量能回答三个问题：可量化检查清单

#### 能否打板（进场前）

- [ ] 板块/情绪周期处于上升或修复期（退潮期一律降低打板预期）；
- [ ] 量比早盘 > 1.5 且竞价量较昨日有效放大（增量资金到场）；
- [ ] 换手处于该流通盘"健康活跃"区间，**非高位天量**；
- [ ] 若打首板：放量换手板优先于缩量一字板（一字仅竞价博成交）；首板换手 25%–40% 为佳，> 50% 视为筹码过度松动；
- [ ] 封板时点封单**持续增厚**、封流比 ≥ 2%（< 1% 视为偏弱封不予参与）；
- [ ] **量价配合**（放量上攻），无明显放量滞涨/背离。
- 否决项：高位天量、放量滞涨、封单反复撤单对倒——出现任一，放弃。

#### 能否持有（持仓中，T+1 隔夜决策）

- [ ] 尾盘封单稳固/增厚，封成比 > 3（> 10 可博一字续板）；
- [ ] 当日换手未冲到该流通盘"见顶高危"区间（连板加速段龙头换手维持 10%–30% 区间、锁仓中换手下降为健康）；
- [ ] 上涨途中回调为**缩量中继**（缩量、不破 5/10 日线），而非放量下跌；
- [ ] 连板序列中本板攻击量未明显小于上一板（无量价背离）。
- 减仓信号：封成比 < 1、封单尾盘大幅缩水、放量炸板未回封。

#### 是否见顶（卖出/规避）

- [ ] 高位放出**天量**（远超前期任意单日 + 换手进入高危档）→ 顶部预警；
- [ ] 价创新高但量未跟上（**量价顶背离**）→ 涨势衰竭；
- [ ] 连续一致缩量后出现**分歧日放量却无法回封**→ 退潮起点；
- [ ] 放量滞涨 / 放量长上影 / 高位巨量炸板 → 主力派发高发结构。
- 处置：满足两项及以上，优先在分歧日冲高、尾盘或次日竞价高开时减仓离场，不恋战。

> 本章特定提示：量能与封单是可被人为操纵的信号——封单可在不可撤单段之前瞬间撤换、量能可由对倒虚增，缩量一字一旦开板因前期零换手而易直接闷杀；因此本章所有阈值只作概率性参考，须结合封单的可撤/不可撤时点与封成比动态变化交叉验证，不可仅凭单一静态量能数值下注。


## 八、筹码分布分析

筹码分布（成本分布，又称"流通股持仓成本分布"）是把历史每个价位的成交量按一定的换手衰减系数逐日叠加，估算出当前全体流通筹码在各价位上的堆积量。它不是实时持仓快照，而是一种基于"每日换手均匀、历史成交按衰减系数移动"假设的统计估算（详见本章末"局限性"）。本章只讲短线视角下怎么用，不展开算法推导。看盘软件中通常以红色表示获利盘、蓝色表示套牢盘，白线为市场平均成本（持仓成本重心），并附带"获利比例""90%成本区间""集中度"等读数。

### 8.1 必须先认清的几个读数

- **获利盘比例**：当前价之下的筹码占比，即"账面浮盈的人占多少"。70%以上为高获利区，30%以下为低获利区，10%以下接近"全员套牢"。
- **平均成本（白线）**：全体筹码的成本重心。股价相对白线的位置，决定多数人是浮盈还是浮亏，直接影响拉升时的抛压性质。
- **90%成本区间与集中度**：90%的筹码所落在的价格区间。集中度数值越小，筹码越集中、控盘越强。常用经验阈值：
  - 集中度 > 30%：筹码发散，分歧大、控盘弱。
  - 15%–30%：中度集中。
  - 10%–15%：中高度控盘，具备波段拉升基础。
  - < 10%：高度控盘，一旦启动幅度往往可观（但需警惕高位高集中=主力已锁定利润待出货）。
- 注意：集中度低本身只说明"筹码集中"，**不区分集中在高位还是低位**，必须结合价格位置一起读，否则容易把高位出货前的"假集中"当成低位吸筹。

### 8.2 五种核心形态与含义

- **低位单峰密集**：筹码在低位一个狭窄价区高度堆积，上方几乎无套牢盘，平均成本贴近现价。代表经过充分换手，浮筹清洗干净、成本趋于一致，是吸筹接近完成、最具向上弹性的形态。
- **高位单峰密集**：形态相似但发生在大涨之后的高位。获利盘普遍丰厚，一旦放量滞涨极可能是派发，**不是买点而是减仓信号**。位置决定性质，与低位单峰天差地别。
- **多峰（上下双峰/多峰林立）**：存在两个及以上密集峰，对应不同时期、不同成本的持有人群。上方峰=套牢盘=拉升阻力；下方峰=获利盘=潜在抛压来源。多峰说明分歧大、没洗干净。
- **上方套牢盘（密集套牢峰）**：被套筹码堆积形成的"盖子"。股价上行到该价区附近，解套盘抛压集中释放，是天然压力位。
- **下方获利盘**：现价之下的获利筹码。比例过高时，浮盈兑现冲动强，拉升中容易引发获利了结砸盘；比例过低（深套）则抛压枯竭但人气也差。

筹码运行的常态是**发散→密集→再发散**的循环：低位发散是派发后的无序，逐步收敛为低位单峰是吸筹，向上突破后筹码上移、再到高位密集则进入派发倒计时。

### 8.3 用筹码做四件具体判断

1. **判断上方阻力（拉升的天花板在哪）**
   - 找现价上方最近、最厚的套牢密集峰，其价格区间就是第一压力带。
   - 量化参考：上方套牢峰若占流通盘 > 20%，需要明显放量（突破日换手通常 > 8%，强势票 > 15%）才能有效吃掉；占比 < 5% 则几乎无阻力，属"光头"上行结构。
   - 若上方完全无峰、空间打开，是连板与主升的有利结构。

2. **判断洗盘是否到位**
   - 洗盘到位的标志：上方套牢峰被消化或下移、下方获利盘被打散，逐步形成**单峰密集**，平均成本上移并贴近现价。
   - 量化参考：调整期间累计换手率达到流通盘的 1–2 倍以上、获利盘从高位回落到 30%–50% 区间、90%集中度收敛（如从 25% 降到 12% 以内），可视为换手较充分。
   - 反例：股价跌但筹码峰几乎不动、集中度不收敛，说明只是缩量阴跌而非换手洗盘，未到位。

3. **判断主力锁仓程度**
   - 低位单峰 + 集中度 < 12% + 上方无套牢峰，指向较强锁仓与控盘。
   - 进一步验证：拉升或调整中若**底部峰始终不发生明显衰减**（筹码没从低位区流出），说明低位筹码被锁定、未交出，锁仓度高；若底部峰随上涨快速消融，则是边拉边出，锁仓弱。

4. **判断打板还是低吸**
   - 偏**打板**的筹码结构：低位单峰密集 + 上方干净（套牢峰极薄）+ 突破当日放量，攻击阻力小，适合在突破/首板逻辑里参与。
   - 偏**低吸**的筹码结构：获利盘已充分换手回落、平均成本上移、回踩不破下方密集峰且缩量企稳，适合在支撑位低吸。
   - 不适合追板的结构：高位单峰高集中、上方密集套牢峰未消化、获利盘 > 80% 且放量滞涨——这类是抛压随时释放的位置。

### 8.4 与连板、反包、低吸买点的结合

- **低位单峰密集突破（连板/首板）**
  - 触发条件：低位单峰密集成形 → 集中度 < 12% → 上方无明显套牢峰 → 突破日放量（换手 > 8%，主板个股以涨停或准涨停封板为佳）。
  - 操作：突破/首板为第一买点；次日若不破前一日密集峰下沿、且承接放量，可作连板跟随。封单要看强度，烂板（反复开板、封单薄）说明分歧大、筹码未真正锁定，降级处理。

- **反包**
  - 适用结构：上涨中途的回踩洗盘，调整后仍是单峰或下方密集峰完好，获利盘回落但平均成本未被击穿。
  - 触发：调整缩量到位（量能萎缩至高峰日的 1/3 以下）后，**重新放量收复前高/反包阳线**当日介入；筹码上看，回踩未引发底部峰大幅衰减是关键过滤条件。
  - 否决：若回踩已把低位密集峰打散成多峰、集中度重新发散，反包失败概率高。

- **低吸买点**
  - 适用结构：获利盘充分换手后的二次低吸——平均成本上移、现价回踩到下方密集峰上沿、获利比例落到 30%–50%。
  - 触发：回踩不破密集峰下沿 + 缩量 + 出现止跌K线（如缩量十字星、长下影），在密集峰区间分批低吸，跌破密集峰下沿一定幅度（如 3%–5%）止损。
  - 量化锚点：以90%成本区间下沿、平均成本线作为支撑与止损参照，比单纯看均线更贴近真实持仓成本。

### 8.5 实战检查清单（介入前逐条核对）

- [ ] 密集峰在**低位**还是高位？（高位单峰一律警惕派发，不当买点）
- [ ] 现价上方最近套牢峰的价位与占比？> 20% 需放量验证，< 5% 视为空间打开。
- [ ] 90%集中度是多少、是否处于**收敛**趋势？（< 12% 为强控盘参考）
- [ ] 获利盘比例多少？打板看上方是否干净；低吸看是否回落到 30%–50%。
- [ ] 平均成本线相对现价的位置，浮盈/浮亏结构是否支持继续上攻？
- [ ] 底部峰在上涨/调整中是否被锁定（不衰减）？以判断锁仓度。
- [ ] 突破/反包当日量能是否配合？（换手、封单强度、是否烂板）
- [ ] 是否已用价量、龙头地位、题材强度等**其它维度交叉验证**？筹码集中是涨的必要非充分条件。

### 8.6 制度边界提示（影响筹码读法的差异点）

（交易制度统一口径见《制度与术语基线》一节，本章只列差异点）筹码"是否换手充分""峰怎么移动"与板幅、ST 切换、新股、T+1 密切相关，套用阈值前先确认标的板块：

- **板幅差异**：±20% 板块（创业板、科创板）单日有效突破的换手要求通常高于主板 ±10% 标的，换手与突破放量的衡量标准随板幅不同而抬高。
- **ST/*ST 切换**：截至本手册撰写日（2026-06-13）主板风险警示股仍执行 ±5%，自 **2026-07-06** 起主板风险警示股涨跌幅由 ±5% 调整为 ±10%（切换日前后以券商行情实际显示的涨跌停价为准）。在该节点前后，ST 股的连板节奏、套牢峰消化速度与放量标准会发生显著变化，跨节点比对历史筹码形态时须区分新旧规则。本战法原则上回避 ST 标的。
- **T+1 跨日换手**：当日买入次日才能卖出，筹码"换手"实际跨日完成；收盘集合竞价 14:57–15:00 全程不可撤单，打板封单、低吸承接均需结合竞价与盘口，不能只看筹码静态图。
- **新股噪声**：新股上市前 5 个交易日不设涨跌幅（北交所新股首日不设限），筹码尚未沉淀，成本分布噪声大、参考意义有限，不宜直接套用单峰/多峰判断。

### 8.7 客观看待：筹码分布是估算，有滞后与局限

- **本质是估算，非真实持仓**：算法基于"每日换手在各价位均匀分布""历史成交按固定衰减系数移动"等假设；衰减系数（常见取值区间 0.1–10）属人为参数，不同软件、不同设置画出的图并不一致，没有唯一正确答案。
- **存在滞后**：筹码移动靠逐日成交叠加，是对历史的回溯统计，对当日盘中的真实买卖、主力对倒、大宗交易、融券、ETF/指数成分调仓等反映滞后甚至失真。对倒放量会"伪造"换手，让筹码看起来已洗盘，实则未必。
- **不识别持有人身份**：筹码图分不清是主力、游资还是散户，"集中"也可能是被动指数资金或一致性套牢，并不等于有主力锁仓拉升的意愿。
- **正确定位**：把筹码分布当作**辅助过滤器与风险位标尺**（看阻力位、支撑位、获利盘结构、控盘强弱倾向），而不是独立买卖信号。任何阈值都需与价量、题材、龙头地位、大盘环境交叉验证。

筹码特有风险提示：对倒放量、大宗交易、ETF/指数成分调仓会"伪造"换手，使筹码图看似已洗盘而实则未必，须以90%成本区间下沿、平均成本线作为止损锚点，而非把"低位单峰密集"当作重仓或博弈连板确定性的理由。


## 九、打板与追板买入逻辑

打板的本质是用确定性的价格（涨停价）去博取后续溢价，代价是放弃了价格区间、只能接受最差成交价。它的盈利来源不是公司基本面，而是次日及后续的情绪溢价（接力资金愿意以更高价承接）。因此打板是一种典型的"高赔率、低胜率、强纪律"博弈：单笔正确时弹性大，单笔错误时常以"一字闷杀""次日低开-7%以上"收场。本章只解决一件事——在什么板数、什么情绪阶段、什么封单结构下值得按下买入键，以及在什么条件下必须放弃。所有数值阈值均为经验区间，须用自己的历史交割单回测校准，不能照搬。

### 9.1 先锚定制度边界（决定打板的物理空间）

打板前必须先确认标的所在板块的涨跌幅与竞价规则，因为它直接决定"封板成本、撤单窗口、烂板概率"。（交易制度统一口径见《制度与术语基线》一节，本章只列差异点）

与打板直接相关的差异点：

- ST/*ST 截至本手册撰写日（2026-06-13）仍执行 ±5%；沪深北交易所已发布修订规则，自 **2026-07-06 起主板风险警示股涨跌幅由 ±5% 调整为 ±10%**（创业板/科创板风险警示股为 ±20%）。切换日前后以券商行情实际显示的涨跌停价为准，介入前务必逐票确认当日实际限幅。本战法原则上回避 ST 标的。
- 新股无涨跌幅阶段（主板/双创前 5 日、北交所首日）不存在"涨停板"，不属于本章打板范畴；该阶段的 30%/60% 两档盘中临停（每次 10 分钟）只适用于这类"无涨跌幅限制"的股票，不是已上市股票的日常机制。
- T+1 约束下，打板买入后至少持仓到次日，隔夜风险无法当日规避——这是本章一切博弈的根本前提。

板块差异对打板的直接影响：±20%/±30% 的双创、北交标的，封板所需资金更大、封单"含金量"更高，但一旦炸板回撤空间也成倍放大（双创炸板可瞬间从 +20% 回落到 +10% 甚至翻绿）。对同等仓位，双创/北交单票止损金额应按更大波动率折半处理。

### 9.2 把买入模式拆成五类（每类的逻辑与适用情绪阶段不同）

不同"板数×位置"对应完全不同的买方结构和容错。先归类，再谈时机。

- 低位首板（地位/低位第一个涨停）
  - 逻辑：股价处于阶段低位（如距离近 60 日高点 ≥20%、或长期横盘后第一个涨停），博弈"从 0 到 1"的预期。胜率相对高、单板弹性有限，是退潮末期/冰点回暖期的主力打法。
  - 适用阶段：情绪冰点回暖、退潮转启动。高潮期不该把仓位押在弱辨识度的低位首板上。
- 二板 / 三板加速（一进二、二进三）
  - 逻辑：博弈连板趋势延续，买方是专门做"晋级"的接力资金。二板是分歧转一致的第一道关，三板是加速档，弹性最大但反包/闷杀风险同步放大。
  - 适用阶段：启动期到发酵期最舒服；进入高潮期后，越高的连板越接近"末班车"，容错急剧下降。
- 中继板（趋势中途的整理后再涨停）
  - 逻辑：强势趋势股回调缩量后重新放量涨停，或龙头在调整 1–3 日后的"修复中继"。买的是趋势延续而非情绪极值，节奏比纯连板缓，适合中军/趋势票。
  - 适用阶段：发酵期、强周期的波段切换点。
- 反包板（前一日大阴线/炸板后次日重新涨停反包）
  - 逻辑：博弈"分歧转一致"。前一日放量炸板或大幅回撤，次日若能重新封板（尤其放量反包并吃掉前一日实体），说明承接资金强、空头被打穿。赔率高，但前提是题材/资金仍在，否则就是下跌中继。
  - 适用阶段：发酵期、龙头分歧后的修复日；退潮期的反包多为诱多，须严控。
- 高度板（高位连板，市场最高板或次高板）
  - 逻辑：博弈情绪龙头的虹吸效应，是市场情绪的"风向标"，但越高位接力越是"博傻末端"。一旦高度板倒下，往往触发退潮，隔夜接力是全场风险最高的动作。
  - 适用阶段：仅在情绪强周期、且自己有明确的次日卖点纪律时参与；高潮当日的最高板尤其要警惕"今天涨停、明天闷杀"。

### 9.3 打板时机：四种动作的取舍与风险

同样是买涨停，进场的"时点选择"决定了你的成本与被埋概率。

- 秒板 / 抢板（开盘后极短时间内主动以涨停价挂单抢封）
  - 适用：开盘竞价已极强（见 9.4 竞价信号）、题材逻辑硬、自己有当日盘前计划的目标票。
  - 风险：抢在封死之前进场，等于赌"它会封住"。若未封死即变成高位站岗。仅用于辨识度最高的核心标的，且单票仓位下调。
- 排板（在涨停价上提前排队，等成交）
  - 适用：判断该票大概率封板、但封单尚不极端，提前排队以争取靠前队列、提高成交概率。
  - 风险：排进去后若封单被抽撤、开板，你的买单会在开板瞬间成交在涨停价，随后股价下杀——"排板被反核"是高位连板最常见的亏损来源。排板必须配合"开板即撤/即砸"的预案。
- 扫板（盘中涨停被打开后，用大单扫掉卖压重新买回涨停）
  - 适用：强势票盘中洗盘式炸板、题材未坏、炸板后快速回封的瞬间，扫掉抛压介入。
  - 风险：分不清"洗盘炸板"还是"出货炸板"。判据：炸板后回封时间（越快越好，理想 <2 分钟）、回封时封单是否快速重堆、炸板期间是否放出巨量对倒。回封无力、反复炸板即放弃。
- 烂板介入（封板过程极其反复、多次开板才勉强收涨停）
  - 取舍：烂板代表分歧极大、主力封板意愿弱或在借涨停出货。原则上不打烂板；唯一例外是"低位首板因获利盘松动而烂、但题材为当日最强主线"，且必须把仓位降到试探级别。高位连板的烂板一律放弃。

时间维度的经验口径：开盘 30 分钟内（约 10:00 前）首次封板的强度，普遍高于午后才封的板；尾盘 14:30 后才涨停、且为当日第一次涨停的票，次日溢价不稳定，作为追板对象优先级最低。临近 14:57 用收盘集合竞价"扫尾板"博次日的，须接受"为封板而封板、次日大概率不接力"的高失败率。

### 9.4 封单强度与真伪评估（打板的核心风控）

封单是打板唯一能实时观测的"资金投票"。评估封单要同时看量、看结构、看动态，三者缺一不可。本节为全书封单评估的权威出处，统一采用"封流比 = 封单金额 ÷ 流通市值"口径，剔除流通盘大小干扰。

- 封流比——衡量"封得牢不牢"
  - 统一分档：**封流比 ≥2% 视为封板较实、抛压可控；1%–2% 一般；<1% 偏弱**——薄封易被砸开，不可视为有效封板。早盘即在涨停价上堆出高封流比，属强势封板。
  - 对照盘子大小：小盘股封单几千万即可达到较高封流比，大盘股需上亿乃至数亿封单才达标。绝对封单额必须除以流通市值看，单看"封单多少亿"无意义。计算时以"自由流通市值"为准，避免被大股东锁仓股名义流通误导，必要时再用"封成比"修正。
- 封成比（封单 / 当日成交额）——衡量"换手是否充分、筹码是否干净"
  - 一字板/秒板：成交极小、封成比极高，筹码锁定好但买不进（只能竞价或排队赌开板）。
  - 换手充分后的封板（如换手 25%–40% 的健康区间后封死）筹码相对干净，次日抛压可控；换手已极高（首板单日换手 >50%、或高位加速段龙头日均换手显著超出 10%–30% 区间）才说明筹码大幅松动，封住也易反包，列入警惕。
- 封单动态——比静态封单量更重要
  - 抽撤：封单从高位快速减少（如 10 分钟内封单缩水 >50%、或封流比跌破 1%），是主力撤离、即将开板的强信号——出现即准备放弃/撤单。
  - 反复开板：当日封→开→封反复 ≥2–3 次，分歧巨大，除"低位最强主线洗盘"外原则放弃。
  - 大单结构：健康封单是多笔中大单堆叠、且封单随时间增厚；若涨停价上仅靠 1–2 笔超大单"独撑"，该单一旦撤离即开板，属脆弱封单；盘口频繁出现"大买单挂出→撤单→再挂"的诱导，视为做盘出货。
- 真假封板的快速判据（满足越多越可信）
  1. 早盘封（约 10:00 前）且封后未再开板；
  2. 封流比 ≥2% 并随时间增厚而非递减；
  3. 题材为当日主线、有板块梯队共振（非独苗）；
  4. 封单大单结构分散、非单笔独撑；
  5. 该票在 9:25 竞价已显强势（竞价量比放大、竞买额占比高、未大幅低开诱空）。

### 9.5 板数 × 情绪阶段 的仓位与容错矩阵

仓位不是固定数字，而是"赔率×胜率×情绪阶段"的函数。情绪阶段判断（冰点/启动/发酵/高潮/退潮）由全市场涨停家数、连板高度、炸板率、跌停家数、昨日涨停今日红盘率等温度指标决定，本章只给出对应到打板的仓位口径。

**矩阵中的百分比为"单票占当日打板可用资金"的上限，并非账户总仓位。** 换算到账户本金后，单票仓位一律不得突破《全局统一口径》的板幅上限：10cm 标的 ≤20%、20cm 标的 ≤10%、30cm 标的 ≤7%（对应单笔账户最大风险约 2%）。即使矩阵上沿显示 20%，对 20cm/30cm 标的也须先按板幅上限折算后再执行，谁更紧用谁。

| 情绪阶段 \ 板数 | 低位首板 | 二板/三板 | 中继板 | 反包板 | 高度板 |
| --- | --- | --- | --- | --- | --- |
| 冰点（不抄底、等回暖） | 0–试探仓（≤10%） | 0 | 0 | 0 | 0 |
| 启动（回暖看新龙） | 轻仓 10%–20% | 轻仓 10%–20% | — | 谨慎 ≤10% | 0 |
| 发酵（主升、可加大） | 中仓 15%–20% | 中仓 ≤20% | 中仓 ≤20% | 中仓 ≤20% | 轻仓 ≤15% |
| 高潮（冷静、防退潮） | 减半 ≤10% | 减半 ≤15% | 减半 ≤15% | 谨慎 ≤10% | 试探/不打 |
| 退潮（快离场、不接力） | 0 | 0 | 0 | 0（多为诱多） | 0 |

容错（隔夜/次日止损口径）配套原则：

- 板数越高、位置越高，次日止损越要"硬且快"。高度板次日若低开 >-3% 或冲高回落破开盘价不创新高，开盘 30 分钟内无条件减/清；首板/低位板可给到次日不破前一日涨停价分时支撑即持有。
- 一笔打板的最大可接受亏损（含隔夜跳空）应在下单前算清：以"假设次日一字跌停/大幅低开"的最坏情形反推单票仓位，确保单笔最坏亏损 ≤ 账户净值的 2%。双创/北交标的因波动翻倍，单票仓位相应折半，且不得突破上述板幅仓位上限。
- 高潮日严禁满仓打最高板隔夜；退潮信号（最高板倒下、炸板率飙升、跌停家数增多）出现后，当日不再新开打板仓。

### 9.6 买入触发条件清单（须同时满足，缺一不打）

只有以下条件全部满足，才允许按涨停价进场：

1. 情绪阶段允许：处于启动/发酵期，或冰点末期明确回暖信号；非高潮末端、非退潮日。
2. 题材逻辑硬：属当日主线/次主线，有板块梯队共振（≥2–3 只同向涨停），非孤立独苗。
3. 位置合适：与所选模式匹配（低位首板距高点有空间；连板未到市场情绪天花板的"末班车"高度）。
4. 封板时间早：约 10:00 前首次封板（中继/反包可放宽，但忌尾盘才封的首封）。
5. 封单达标：封流比 ≥2%，且封单随时间增厚、未见抽撤。
6. 封单结构健康：多笔中大单堆叠、非单笔独撑；当日开板次数 ≤1 且快速回封。
7. 换手合理：首板换手在 25%–40% 健康区间、未超过 50% 的过度松动线（缩量一字除外）；连板加速段龙头日均换手处于 10%–30% 区间，未异常放大。
8. 竞价不背离：9:25 竞价显强（量比放大、竞买额占比高、未恶性低开诱空）。
9. 风控就位：已设定次日止损价、单票仓位与最坏亏损额，且符合 9.5 矩阵及板幅仓位上限。

### 9.7 放弃 / 不打条件清单（命中任意一条即放弃）

以下任一情形出现，立即放弃排板、撤回未成交买单、不追：

1. 情绪退潮：最高板倒下、炸板率 >50%、跌停家数明显多于涨停家数、昨日涨停今日红盘率 <50%。
2. 封单抽撤：封流比快速下滑（如 10 分钟内缩水 >50% 或跌破 1%），或封单由增转减。
3. 反复炸板：当日开板 ≥2–3 次的烂板（低位最强主线洗盘除外，且仅试探仓）。
4. 单笔独撑：涨停价仅靠 1–2 笔超大单封板，或盘口出现挂撤诱导式做盘。
5. 位置过高：连板已到市场情绪天花板的"末班车"高度、且无强逻辑支撑的纯博傻高度板。
6. 换手异常：首板单日换手 >50%（筹码过度松动），或高位加速段龙头换手大幅偏离 10%–30% 区间。
7. 封板过晚：尾盘（14:30 后）才首次涨停的弱封，作为隔夜接力对象。
8. 竞价背离：核心票 9:25 竞价大幅低开、放量诱空，或主线集体竞价走弱。
9. 逻辑证伪：题材证伪（利好兑现/被官方/媒体证伪）、独苗无梯队、或板块已分歧瓦解。
10. 风控不满足：算出的最坏隔夜亏损 > 账户净值 2% 上限，或当日打板仓位、单票板幅仓位上限已用满。

### 9.8 本章特定风险提示

除全书第一章已统一声明的打板高风险属性外，本章须额外强调一条具体风险：**排板与扫板在封单被瞬间抽撤时，未成交买单会成交在涨停价随后立刻下杀（"被反核"），这是连板打板独有的、止损也难以在理想价位兑现的结构性亏损来源**——凡排板、扫板，必须预先备好"开板即撤单/即砸"的执行预案。


## 十、接力、低吸与半路买点

直接打板是用涨停价主动买入、追求"封单确认 + 次日溢价"的最锋利打法，但它把买点定在情绪与价格的最高处，一旦次日不及预期，回撤幅度等于"涨停溢价 + 高开兑现"的全部空间。接力、低吸、半路这三类买法本质上都是在**降低买入位置、换取容错空间**：用更早或更低的进场点，换取"封不上/竞价不及预期"时更小的止损,代价是确定性更低、需要更强的盘口与情绪判断。本章给出每种买法的触发条件、买点定位、止损位、适用情绪阶段与仓位区间，并贯穿"分歧—一致—弱转强"这条主线。

（交易制度统一口径见《制度与术语基线》一节，本章只列差异点）本章战法的差异点仅在于：竞价接力、弱转强、低吸的具体战术都建立在集合竞价时点与各板涨跌幅之上，跨板块操作必须先换算对应板幅（主板 ±10%、创业板/科创板 ±20%、北交所 ±30%）再设止损与点数口径。

### （一）三种状态的识别：分歧、一致、弱转强

这是所有买点的"地基"。买在哪种状态、做哪个方向的转换，决定了盈亏比。

- **一致**：资金方向高度同向。盘口特征为板块批量涨停、前排标的缩量加速、封单厚、抛压极小、涨停时间早（多在 10:30 前）。一致是情绪最热、也是风险最高的时点——一致日买入意味着"接最后一棒"的概率上升。
- **分歧**：资金对该方向出现分歧。特征为高位放量、断板、炸板、冲高回落、涨停反复打开封不死、龙头竞价高开后回落。分歧本身不可怕，关键看次日是"分歧转一致"（资金回流、回封）还是"分歧转崩溃"（资金离场、低开走弱）。
- **弱转强**：标的或板块在经历相对弱势（炸板、烂板、断板、尾盘封、缩量调整）后，次日竞价或盘中重新走强的过程。它是"分歧转一致"在个股层面的具体表现，是接力与低吸最核心的买点来源。

可量化的状态判定（用于触发任何买入前的前置过滤）：

- **转强成交量确认**：转强当日成交量需较前 5 日均量放大 ≥30%，否则视为缩量假强、降级处理。
- **分歧转一致的有效信号**：分歧次日竞价高开（参考 +2% 以上），或盘中 10:30 前重新回封；二次分歧仍能回封，强度更高。
- **一致转分歧的离场信号**：一致/高潮日出现爆量滞涨、涨停反复打开、龙头炸板不回封——这是减仓而非加仓的信号。
- **纪律边界**：同一强势股的"分歧转一致"通常不超过 2—3 次，超过后溢价衰减、风险收益比恶化；分歧当天原则上不打板、不追高（低吸除外，见下）。

> 操作总则：**买"分歧转一致"的次日早盘，卖"一致转分歧"的当天爆量**。接力与低吸是这条总则在不同价位上的落地。

### （二）次日接力：高位接力、弱转强接力、竞价接力

接力指次日继续买入昨日的强势/连板标的，赌情绪与资金的延续。它比打板更早介入次日的换手承接，核心风险是"昨日的强不等于今日的强"。

#### 1. 高位接力（连板换手承接）

- **适用情形**：主升期、主线明确、龙头连板且板块有 2 个以上同步跟风的"卡位"阶段。
- **量能触发**：昨日涨停应是**放量涨停**（新资金进场、换手充分），优先接力；昨日缩量一字/缩量死板说明筹码锁死、无承接,反而风险高（次日一旦撬板易吃面）。
- **买点定位**：次日竞价不破昨收、平开至小幅高开，开盘后分时不破水位、回踩分时均线企稳放量上攻时介入；不追竞价直接顶上去的高开大长腿。
- **止损位**：跌破昨日涨停价（即昨日实体上沿）或盘中放量滞涨、分时三波不过前高时离场；硬止损以亏损 5%、且当日不回封为兜底。
- **情绪阶段与仓位**：主升期可上核心仓（如 3—5 成集中于龙头一只），分歧期降至 3 成以内，退潮期不做高位接力。

#### 2. 弱转强接力（分歧后的回封承接）

这是接力里盈亏比最优的一类，专做"昨日弱、今日强"。

- **连板弱转强**的识别：标的是板块最高空间板或核心卡位、板块内仍有 2 板以上跟风、昨日属于炸板/烂板/尾盘封等"弱"形态，而次日分歧日能在 **10:30 前回封**。
- **买点与加仓节奏**：首次分歧回封轻仓试错（约 20%）；若次日竞价再度爆量、高开走强（二次确认），再加仓。不在第一次炸板的下杀里盲目接。
- **止损位**：跌破前日涨停价，或回封后再度炸板不回、放量滞涨即离场；统一以"跌破转强 K 线/分时实体 1/2 或亏损超 5%，3 分钟内无条件离场"为铁律。
- **情绪阶段**：最适合**分歧期向主升期的切换点**，即退潮后的第一次弱转强、或主升中途的洗盘回封。

#### 3. 竞价接力（集合竞价定方向）

利用 9:15—9:25 集合竞价的量价信息提前定位强弱，是接力中最依赖盘口、容错最低的一种。

- **竞价识别条件**：
  - 9:15—9:20 因可撤单，常见大单虚拍顶涨停制造繁荣假象，**不可据此追单**；重点看 9:20—9:25（不可撤单）阶段的真实承接。
  - 弱转强的有效竞价盘口：竞价量比逐级抬高、价格阶梯上行（量价齐升）、临近 9:25 仍有大单净流入，且**竞价成交额 ≥ 昨日全天成交额的 15%**、高开幅度 ≥ 2%（板块/题材越热门，阈值可适当上调）。
  - 警惕"高开后竞价跳水"：9:20 后价格逐级走低、买单撤离，多为出货式高开，放弃。
- **买点定位**：以 9:25 集合竞价成交价直接介入，或开盘后不破竞价低点、分时第一波放量上攻时跟进。
- **止损位**：开盘即破竞价低点、或高开低走跌破昨收，分时不能快速收回时止损；硬止损同样 5% 兜底。
- **情绪阶段与仓位**：竞价接力适合主升期与分歧转一致的确认日，因介入点高、确定性靠竞价"赌"出来，单笔仓位建议控制在 2—3 成，不宜重仓。

### （三）低吸：四种回踩买点

低吸是在强势股的关键支撑位、用低于涨停的价格买入，目标是拿到最低成本筹码、获得最大容错空间。它最适合**调整期、分歧期、以及一致后第一次缩量回踩**——而非情绪最热的一致日。低吸的共性纪律：**只低吸真核心/真龙头，不低吸跟风伪核心**（伪核心的支撑位极易跌破）；**分歧当天不盲目低吸接刀**，等支撑企稳信号出现再动手。

#### 1. 分时回踩低吸

- **适用情形**：盘中放量上攻后的自然回踩，主升期内的强势震荡。
- **买点定位**：回踩分时均线（黄线）或回踩至放量起涨点附近、缩量不破、再度放量上翘时介入；回踩幅度参考不破当日分时水位线（如不破开盘/不破前一波低点）。
- **止损位**：跌破回踩低点或分时均线持续下方运行即走，单笔亏损 2%—3% 即可止损（位置低，止损也应更紧）。

#### 2. 均线低吸（5 日/10 日线）

- **适用情形**：连板后第一次缩量调整、强势股回踩短均线的 N 型走势。
- **买点定位**：跌回 5 日线（强势股）或 10 日线（题材龙头回调），且**不跌破前一日涨停价**，收阳企稳时低吸；趋势型标的要求 20/30 日均线仍向上、调整期缩量至均量 60% 以下。
- **止损位**：有效跌破 5 日线（或对应均线）、或跌破前日涨停价；硬止损 5%。

#### 3. 反包前低吸（阴线后的反包预埋）

- **适用情形**：强势股放出一根分歧阴线（断板/冲高回落）后，次日预期资金回流"反包"。
- **买点定位**：次日竞价高开 > 2%、竞价成交额 ≥ 昨日总额 15% 时，于竞价最后 1 分钟或开盘后突破前一日阴线实体中点（1/2 位）时介入。
- **止损位**：若低开跌破前一日阳线/阴线实体 1/2，3 分钟内止损；不反包就不恋战。

#### 4. 跌停板/翘板低吸（高风险、仅限龙头）

- **适用情形**：高位龙头次日竞价低开甚至一度跌停，但有资金在跌停板上撬板、翘起来（弱转强的极端形态）。这是低吸里风险最高的一类，**只在确认是板块绝对核心、且有真实大单撬板时才做**。
- **买点定位**：跌停板上出现持续大单扫板、封单被快速撬开、分时翘头放量时少量介入；严禁在缩量阴跌的跌停板上盲目抄底。
- **止损位**：翘板失败、重新回封跌停或封死不开即认错（T+1 下次日才能卖，故位置与仓位必须从严）。仓位压到 1—2 成试错，把它当"赔率交易"而非常规买点。

### （四）半路：盘中未封板时上车

半路是在个股尚未涨停、距涨停约剩 2—3 个点时介入，跟随主力完成最后的拉升封板。它介于打板与低吸之间：成本低于打板（封板后即有浮盈），未封板时的亏损也小于追在涨停价被炸板。

- **适用情形**：震荡期、题材发酵当天、盘口已有明确放量上攻意图但尚未一致到涨停的阶段。
- **买点定位与触发条件**：
  - 距涨停约 3 个点（主板口径；20cm 板可放宽至距涨停 5—6 个点的有效攻击段）。
  - 早盘放量上攻、分时持续放量、题材有持续热度（板块有跟风共振）。
  - 分时不破前一波低点、量能逐级放大、买盘主动性强（外盘明显大于内盘）。
- **止损位**：上攻失败、放量滞涨封不上板、分时跌破介入点的前一波低点时立即离场；半路的核心纪律是**封不上就走**，不被动持有到收盘等"明天"。
- **情绪阶段与仓位**：半路适合中期/震荡期的折中进攻，单笔 2—3 成；情绪一致的主升期更宜用接力，退潮期半路成功率低、应回避。

### （五）四类买法的风险收益对比与选择

| 买法 | 介入位置 | 容错（止损空间） | 确定性 | 主要风险 | 最适情绪阶段 | 单笔仓位参考 |
|---|---|---|---|---|---|---|
| 直接打板 | 涨停价（最高） | 最小（次日溢价全回吐） | 封单确认，最高 | 炸板、次日低开兑现 | 启动期、主升期龙头加速 | 试错 2—3 成 |
| 竞价接力 | 高开价 | 较小 | 靠竞价量价"赌"出 | 高开低走、虚拍诱多 | 主升期、分歧转一致确认日 | 2—3 成 |
| 高位/弱转强接力 | 次日盘中 | 中等（破昨涨停价止损） | 中等 | 昨强今弱、回封后再炸 | 分歧转主升的切换点 | 弱转强 20% 起，逐步加 |
| 半路 | 距涨停 2—3 点 | 中等（封不上即走） | 中等 | 拉升失败、放量滞涨 | 震荡期、题材发酵当天 | 2—3 成 |
| 低吸 | 支撑位（最低） | 最大（位置低、可设紧止损） | 最低（赌回封/反包） | 支撑跌破、接刀、伪核心 | 调整期、分歧期、缩量回踩 | 试错为主，翘板类 1—2 成 |

选择口径的几条硬规则：

1. **位置越高、止损空间越小、对确定性的要求越高**。打板要求封单与题材确定性最高；低吸位置最低、容错最大，但确定性最低、最依赖"是否回封/反包"的判断，因而单笔以试错仓为主。
2. **情绪阶段决定打法**：启动期用打板抢首板，主升期用接力/竞价接力吃延续，分歧期/调整期用低吸捡便宜，震荡期用半路折中，退潮期空仓——不要在退潮期用任何一种买法去"抄强势股"。
3. **统一止损纪律**：无论哪种买法，跌破"买入所依据的那根 K 线/分时结构实体 1/2"或单笔亏损达 5%，即在数分钟内无条件离场；位置越低（低吸、半路），止损应越紧（2%—3%）。
4. **对追板/打板保持客观**：打板并非天然高胜率，它把买点放在情绪与价格双高处，胜率高度依赖情绪阶段与题材质地；脱离主升期的"逢板就打"长期负期望。接力、低吸、半路的价值，正在于用更低的进场位置改善盈亏比，而不是提高交易频率。

落地检查清单（任一买点下单前逐条确认）：

- [ ] 当前处于哪个情绪阶段（启动/主升/分歧/退潮）？该阶段是否匹配这种买法？
- [ ] 标的是真龙头/真核心，还是跟风伪核心？板块是否有 2 个以上同步标的？
- [ ] 是"分歧转一致/弱转强"还是"一致转分歧"？方向是否站在转强一侧？
- [ ] 量能是否达标（转强日放量 ≥30%、竞价额 ≥ 昨日 15%、高开 ≥2% 等对应阈值）？
- [ ] 买点对应的止损位是否明确、能否承受（破昨涨停价 / 破实体 1/2 / 亏 5%）？
- [ ] 仓位是否与确定性匹配（确定性低的低吸/翘板用试错仓，主升核心才上重仓）？
- [ ] T+1 下，若次日不及预期，能否承受隔夜回撤？跨板块是否已按对应板幅（±10/20/30%）换算止损？


## 十一、卖出与止盈止损逻辑

短线（尤其是打板、追板）的盈亏不在买点，而在卖点。买入决定你"参与什么"，卖出决定你"赚多少、亏多少"。本章给出可执行、可量化的持股周期判断、止盈止损触发条件、分批减仓规则与最坏情况预案。所有数值阈值均为决策锚点，需结合当时的市场情绪周期、个股位置（首板/低位连板/高位连板）与所属板块涨跌幅制度做校准。

### 11.1 先校准制度边界（决定你卖点的"物理上限"）

卖点判断必须先确认标的所在板块的涨跌幅与停牌制度，否则同一套"破板""炸板"标准会失效。（交易制度统一口径见《制度与术语基线》一节，本章只列差异点。）

本章与卖点直接相关的差异点：

- **涨跌幅越宽，卖点纪律越要前移**：双创 ±20%、北交所 ±30% 的单日波动与回撤远大于主板 ±10%，止损要相应放宽绝对百分比、但收紧"破位即走"的执行速度；下文凡未标注板块的百分比阈值均以主板 ±10% 为基准，双创/北交所按板幅比例放大。
- **ST 标的的卖点处置**：主板风险警示股截至本手册撰写日（2026-06-13）仍执行 ±5%；沪深北交易所已发布修订规则，自 **2026-07-06** 起主板风险警示股涨跌幅由 ±5% 调整为 ±10%，创业板/科创板风险警示股为 ±20%。切换日前后以券商行情实际显示的涨跌停价为准。本战法原则上回避 ST 标的，若持有则按其实际涨跌停幅度套用本章破板/止损口径。
- **新股阶段无"破板"概念**：新股不设涨跌幅的窗口内（主板/双创前 5 个交易日、北交所新股首日），只能用分时结构和盘中临时停牌后的竞价定夺，不能套用连板/破板逻辑。
- **T+1 决定处置时点**：你今天买的票最快明天才能卖，**所有"次日竞价定夺"都是对昨日持仓的处置**；减仓挂单要在 **9:20 前**完成布局（9:20—9:25 不可撤单），尾盘走人要在 **14:57 前**下手（收盘集合竞价 14:57—15:00 全程不可撤单）。

### 11.2 持股周期的总框架：你是哪一类持仓

卖点逻辑因"买的是什么"而不同，先归类再套规则：

- **情绪启动期的首板/低位首板**：博弈次日溢价，**默认持股周期 1—2 天**，目标吃次日竞价高开与早盘冲高。
- **低位连板（1 进 2、2 进 3）**：博弈加速与高度，周期 2—4 天，核心看"分歧—一致"切换是否延续。
- **高位连板（5 板以上 / 总涨幅已大）**：博弈情绪溢价的尾段，**周期以"天"甚至"半天"计**，任何减速信号优先于贪婪。
- **趋势/卡位低吸**：周期可放宽到数日至十数日，用均线和趋势结构管理，不适用打板的分时硬纪律。

判断口径："位置越高、连板越多、换手越大、距离你成本的浮盈越厚，持股周期越短、减仓越靠前、止损越严"。

### 11.3 止盈逻辑：分位置、分信号、分批

止盈的核心矛盾是"溢价兑现"与"高度想象"的取舍。规则是 **用确定性的分时/竞价信号触发减仓，用想象空间决定是否留底仓**，而不是凭感觉拿。

#### 高度板止盈（高位连板的减仓节奏）

高位连板的封板是"借情绪兑现"，一旦情绪退潮，封单会瞬间瓦解。处置原则：

- **不在涨停板上加仓，只在涨停板上减仓**：高度板（如 5 板以上）次日竞价高开时，优先在竞价或开盘第一时间减 1/3—1/2，落袋为安，剩余仓位才博弈进一步加速。
- **看封单质量决定去留**：以封流比（封单金额 ÷ 流通市值）衡量，<1% 为偏弱封、且尾盘才勉强封板的，视为弱封，次日不博加速、竞价就走；封单反复被砸薄、封流比持续走低同样按弱封处理。
- **盘中"开板—回封"次数**：高位票当日开板 ≥ 2 次或单次开板后 30 分钟内未回封，视为分歧加剧，破板即减。
- 留底仓的唯一理由：板块尚处主升、龙头地位明确、换手健康（连板加速段龙头日均换手 10%—30% 优于一字闷板的"虚高度"）。底仓不超过原仓位 1/3，且次日竞价不达标立即清。

#### 加速末期出货（识别"最后一棒"）

加速段是利润最厚、也是最危险的阶段。**加速末期的典型特征即为出货信号**，出现即减仓而非加仓：

- 连续缩量一字 / 闷板后突然出现**天量换手板**（换手率较前几日数倍放大、明显突破筹码过度松动的 50% 一线），且盘中宽幅震荡：这是场内资金借场外资金接力出货，分时一旦走弱即减。
- 涨停后**反复开板、放量滞涨**（量能创近期新高但价格滞涨、上影长）：减速信号，破前一波涨停回封低点即走。
- 加速段出现**长上影、放量阴线、涨停被大单砸开后封不回**：视为见顶信号，T+1 次日不抱"再冲一次"幻想，竞价或早盘第一波冲高坚决出。
- "钓鱼/拉高出货"分时：早盘急拉冲板诱多、随后阴跌，分时几乎不回到均价线上方——按 11.4 的分时破位纪律执行。

#### 分时破位 / 破板卖出（盘中持仓的硬纪律）

针对盘中持有的强势票，用分时结构设硬止盈线，触发即执行，不二次确认：

- **跌破分时均价线（黄线）且不能快速收回**：3—5 分钟内未收回均价线，减 1/2；持续运行在均价线下方，清。
- **涨停破板**（封单被打开）：
  - 低位首板/低位连板破板：观察是否 5—10 分钟内放量回封；放量回封可留，缩量回封或封不回则减。
  - 高位连板破板：默认先减后看，不赌回封。
- **关键分时支撑跌破**：跌破当日重要平台/前高/分时颈线并伴随放量，按破位幅度减仓，破位 > 3%（主板）/ > 6%（双创）清仓。
- 主升趋势票可用更慢的级别（如跌破当日 15 分钟低点、跌破前一日涨停实体），避免被分时噪音震出。

### 11.4 次日竞价定夺（对昨日连板/强势持仓的标准动作）

次日 9:15—9:25 的竞价，是连板票最重要的决策窗口。**竞价表现 = 隔夜场外资金的真实态度**，按高开/低开/平开三类定标准：

#### 高开冲高减仓

- **理想高开区间**：高开幅度落在 **+3%—+6%**（主板口径；双创相应放宽到约 +5%—+10%），且竞价成交量占昨日全天成交额约 **8%—15%、越接近 10% 越健康**，说明承接积极、又未过度透支。
- 高开 +6%—+9%（接近涨停）但**竞价量能异常放大**（占比远超 15%）：警惕一致性见顶、获利盘集中出逃，**开盘第一波冲高即减 1/2 以上**，不追板。
- 高开但开盘后 **未能延续上攻、快速回落破均价线**：视为高开低走前兆，减仓优先级最高。
- 操作：减仓挂单在 **9:20 前**布好（9:20 后不可撤），按高开幅度梯度成交。

#### 低开破位走

- 低开本身即弱势信号。**低开幅度 > -3%（主板）/ > -5%—-6%（双创）**，或低开后竞价量能稀少、承接缺失：开盘即按市价或跌停队列前列挂单出，不等反弹。
- 低开后若分时反抽到均价线/昨收附近放量不过：反抽即是减仓点，不补仓、不"等回本"。
- 唯一可短暂观察的例外：板块整体低开（龙头集体低开带动）、个股低开幅度小于板块均值且分时迅速翻红——可给 10—15 分钟观察窗，未翻红即走。

#### 平开看分时

- 平开（约 -2%—+2%）是多空分歧最大、最需要看分时的情形，按盘口承接定夺：
  - 开盘后 **15 分钟内放量上攻、站稳均价线、有效拉升**：持有，按 11.3 分时纪律管理；
  - 开盘后**滞涨、量能萎缩、均价线下方运行**：减仓；
  - 平开后**直接走弱破位**：按低开逻辑处理，破位即清。
- 平开是"给一次机会、不给第二次"——给定的观察窗口（如 10—15 分钟 / 跌破开盘价某一幅度）一到即按结果执行，不无限期等待。

### 11.5 炸板应对（封板后被打开的处置）

"炸板"是当日封住涨停后又被打开。是否处置取决于位置、量能和回封能力：

- **判断三要素**：① 位置（高位炸板比低位炸板危险）；② 量能（炸板放量、换手率较近期均值数倍放大→出货嫌疑大）；③ 回封能力（能否快速放量回封）。
- **低位/低换手炸板**：若快速回封、缩量洗盘性质，可持有或视为低吸点；若反复炸板、封不回，减仓。
- **高位炸板**：默认按出货对待——炸板放量且 30 分钟内未回封，减 1/2 以上；尾盘勉强回封、封单薄弱，次日竞价不博加速、直接走。
- **持仓止损线**：以炸板当日的**分时低点（或破板后的回封失败低点）**作为止损位，跌破即清。
- **次日延续**：炸板票次日若低开或弱势反抽，坚决离场，止损设在炸板日最低价。

### 11.6 放量滞涨与分歧日卖点

分歧日往往是连板节奏的"中继或终结"分水岭，卖点要看分歧能否被资金"转一致"。

- **放量滞涨**：当日量能创近期新高、但涨幅明显收窄、收上影或冲高回落——典型滞涨。处置：
  - 高位放量滞涨：见顶概率高，减仓优先，破当日上攻起点即清。
  - 低位放量滞涨：可能是换手蓄势，给一次观察，但若次日竞价不能延续强势即走。
- **分歧日（如连板途中出现首次大幅震荡 / 一字转换手）**：
  - 资金"分歧转一致"成功（次日竞价抢筹、高开冲板、放量上行）：持有；
  - "一致转分歧"且无承接（次日低开 / 平开走弱、滞涨）：按 11.4 竞价标准减仓离场。
  - 量价口径：分歧日成交量较前一封板日**异常放大数倍**而价格不创新高，倾向兑现而非加仓。
- **核心纪律**：分歧不是加仓理由，"分歧—一致"的确认要由次日竞价与盘口给出，事前不预设、不赌方向。

### 11.7 止损逻辑：跌破成本与关键均线

止损是短线生存的底线，必须先于"止盈想象"执行。设置原则：买入即明确止损位，触发无条件离场。

- **固定百分比止损**：
  - 主板个股，单笔浮亏触及 **-3%—-5%** 即考虑止损，**-7% 为强制止损线**（接近跌停，必须在跌停前出，不与跌停队列赌）；
  - 双创个股（±20%）放宽到 **-6%—-10%** 区间设强制线，但因波动大须更快执行；
  - 打板/追板买入：以**买入当日封板的"破板不回封"**为第一止损条件，破板缩量不回封即按市价走，不等收盘。
- **跌破成本价**：T+1 次日若直接跌破你的买入成本且分时走弱，视为参与逻辑证伪，竞价或早盘即止损，不"等解套"。
- **跌破关键均线**：
  - 短线生命线常用 **5 日均线（MA5）**：强势短线票收盘有效跌破 MA5（且非分时假摔）→ 减仓或离场；
  - 跌破 **MA10**：趋势走坏信号，清仓；
  - 跌破"启动 K 线"实体下沿 / 涨停板起涨点：参与逻辑破坏，止损。
- **量价背离止损**：放量滞涨、放量阴线、量增价跌——即使未触及固定止损线，也应主动减仓，不等价格走到止损位才被动出。
- 止损执行优先级高于一切技术幻想：止损位是规则，不是参考。**当日触发当日执行；T+1 限制下，预判到次日大概率破位的，今日尾盘即处理。**

### 11.8 连板票次日的标准处理流程

把前面拆散的标准合成一条对"昨日连板持仓"的可执行流程：

1. **盘前（昨晚收盘后—今晨）**：复盘封单强弱、板块梯队是否健康、龙头是否仍在、有无利空。预设三套预案（高开 / 低开 / 平开各对应一个动作），写下减仓比例和触发价。
2. **9:15—9:20 竞价观察**：看个股竞价方向与量能，看龙头与板块是集体高开还是分化。**9:20 前完成减仓挂单布局**（高开按 11.4 梯度减、低开挂前列卖出）。
3. **9:25 竞价定价**：
   - 高开 +3%—+6% 且量能健康：减 1/3—1/2，留仓博加速；
   - 高开过大 / 量能异常：减 1/2 以上甚至清；
   - 低开破阈值：开盘即走；
   - 平开：进入分时观察流程。
4. **9:30—10:00 分时验证**：按 11.3 分时纪律——站稳均价线、放量上攻则持有；破均价线不回收、滞涨则继续减。这一时段是当日多空最关键窗口。
5. **盘中维护**：破板按位置处理（低位看回封、高位先减）；炸板按 11.5；放量滞涨按 11.6。
6. **尾盘（14:30—14:57）决断**：未能封板、走弱、量能透支的，**14:57 前**了结，不留隔夜风险；趋势仍强、缩量回踩到位的可留底仓。
7. **记录**：成交后记录触发信号与执行结果，用于校准下次的阈值。

口径：连板越高、浮盈越厚，第 3 步的减仓比例越大、第 4 步的容错时间越短。

### 11.9 最坏情况纪律（一字跌停 / 无法卖出的预案）

短线最大的风险不是浮亏，而是"想卖卖不掉"。必须事前建立预案：

- **一字跌停（开盘即封死跌停、无法成交）**：
  - 立即以**跌停价挂单排队**，并按时间优先尽量靠前（开盘集合竞价即挂）；
  - **不撤单、不向上改价等反弹**——一字跌停意味着流动性枯竭，等反弹通常导致连续跌停损失扩大；
  - 若次日仍一字跌停，继续跌停价排队，直至打开能成交的第一时间出掉（哪怕开板瞬间放量也要走，不赌二次回封）。
- **盘中跌停（非一字，盘中触及跌停）**：在跌停封死前的最后流动性窗口出，不与跌停板封单赌回封；一旦封死，转入"跌停排队"纪律。
- **流动性预案（事前）**：
  - 仓位与个股日均成交额匹配，避免持仓占比过大导致出逃即砸盘；高位、低流通、高换手票尤其要控制单票仓位；
  - 对有退市风险、立案调查、问询、商誉/财务暴雷可能的标的，**绝不参与隔夜**，从源头规避一字跌停场景；
  - 对前期连续涨停（情绪票）持仓，提前把"次日大幅低开/一字跌停"作为基准情景演练，确保单次最坏损失不超过总资金可承受额度（10cm 标的单票 ≤20%、20cm 标的 ≤10%、30cm 标的 ≤7%，对应单笔账户最大风险约 2%）。
- **总仓位纪律**：任何单票的最坏情况都不应危及账户存续。**用仓位上限和分散，确保"卖不出"时也死不掉**——这是比任何卖点技巧都更优先的底层纪律。

**核心检查清单（卖出前自检）**

- [ ] 标的板块涨跌幅与停牌制度是否确认（差异点见 11.1，统一口径见《制度与术语基线》）？
- [ ] 当前持仓属于哪一类（首板 / 低位连板 / 高位连板 / 趋势），对应持股周期与减仓节奏是否清晰？
- [ ] 买入时是否已写明止损位（固定百分比 + 关键均线 + 破板条件）？
- [ ] 次日竞价的高开/低开/平开三套预案是否在 9:20 前布好挂单？
- [ ] 是否出现加速末期信号（天量换手、放量滞涨、长上影、炸板封不回）？出现即减。
- [ ] 是否跌破分时均价线 / MA5 / 成本价并放量？触发即按规则执行，不二次确认。
- [ ] 最坏情况（一字跌停）预案是否就位，单票仓位是否在按板幅设定的上限内、单笔最大亏损是否在账户可承受范围内？

## 十二、龙虎榜与游资席位

龙虎榜是交易所每个交易日盘后公布的"交易公开信息"，把触发异常波动标准的个股，连同当日买入、卖出金额最大的若干家会员营业部（席位）及其具体金额一并披露。它是A股市场少数能直接观察"主力资金身份与方向"的公开数据，也是短线游资战法判断接力意愿、识别出货风险的核心工具。本章只讲怎么读、怎么用、怎么避雷，不重复涨跌停制度本身。

### 上榜规则：触发条件与披露口径

龙虎榜披露分两类口径——**单日异动**（当日盘后即公布）与**连续异动**（连续3个交易日累计触发）。各板块阈值不同，2024年以来沪深交易所推行差异化披露（交易制度统一口径见《制度与术语基线》一节，本章只列差异点），规则按板块区分如下（以收盘后公开信息为准）：

- **沪深主板**：单日触发任一即上榜——日涨跌幅偏离值达 ±7%、日换手率达 20%、日价格振幅达 15%；连续异动为连续3个交易日涨跌幅偏离值累计达 ±20%。
- **创业板 / 科创板**：单日触发任一即上榜——收盘涨跌幅偏离值达 ±15%、日换手率达 30%、日价格振幅达 30%；连续异动为连续3个交易日涨跌幅偏离值累计达 ±30%。
- **ST / *ST**：主板风险警示股截至本手册撰写日（2026-06-13）仍执行 ±5%，自 2026-07-06 起由 ±5% 调整为 ±10%（创业板/科创板风险警示股为 ±20%），切换日前后以券商行情实际显示的涨跌停价为准；龙虎榜阈值显著收紧，连续3个交易日收盘价涨跌幅偏离值累计达 ±12% 即触发披露。这类票本身博弈空间小、流动性差，上榜更多是风险提示而非机会，本战法原则上回避 ST 标的。
- **北交所**：异动与披露标准更宽，单日波动天然更大，**不能套用主板的偏离值直觉**去解读其席位强度。
- **无价格涨跌幅限制证券**（如新股上市前5个交易日、退市整理期等）：交易所披露当日买入、卖出金额最大的前5家营业部及其金额；这类标的波动剧烈，席位数据噪声大。

实操要点：

1. **同一只票当天可能因多个条件同时触发**（如既涨停又换手超标），但榜单只公布一次，席位是合并口径。
2. **披露时间**通常在收盘后当日 16:30—17:30 之间，由各行情终端（东方财富、同花顺等）同步。**当晚做次日决策时务必等正式榜单出来再下结论**，盘中"预测上榜"不可靠。
3. 主板取**前5席**买卖营业部，部分连续异动情形披露口径与单日略有差异，读数时先确认是"单日榜"还是"3日累计榜"——两者解读逻辑完全不同。

### 买卖前五席位结构解读

榜单核心是两张表：**买入前五（买一到买五）** 和 **卖出前五（卖一到卖五）**，各列出营业部名称、买入额、卖出额。读榜的第一步永远是算清楚以下四个量：

- **席位净额** = 该席位买入额 − 卖出额。判断单个游资是真买还是边买边出。
- **榜单合计净买入** = 买五合计 − 卖五合计。注意：这只是"上榜部分"的净额，不等于全场主力净额。
- **净买入占成交比** = 榜单合计净买入 ÷ 当日成交额。衡量上榜资金对当日盘面的真实控盘度。
- **买一占比** = 买一席位买入额 ÷ 当日成交额。判断是否存在"单一大资金主导"。

结构化判断清单：

1. **买盘集中度**：买一明显大于买二到买五之和，说明有单一强资金主导，方向性强、次日确定性相对高；买盘平均分散在5个席位，说明是"多路游资共振"，人气足但分歧也大。
2. **买卖力量对比**：榜单合计净买入为正且占成交比 > 5%—10%，属强势；净买入为负（卖盘 > 买盘）即便股价涨停，也说明**有大资金借涨停出货**，是典型分歧信号。
3. **席位"含金量"**：知名游资 / 机构席位上榜 ≫ 普通营业部上榜。同样的净买入额，顶级游资买一的指示意义远高于无名营业部。
4. **买卖席位是否重叠**：同一营业部同时出现在买方和卖方，需高度警惕对倒（见下文）。

### 知名游资席位与风格流派

席位本质是"营业部账户"，同一游资可能用多个营业部下单，市场通过长期跟踪把营业部与游资风格对应起来（标识非官方、会漂移，需动态更新）。按操作风格大致分四类流派：

- **打板 / 接力型（追涨封板）**：以"炒股养家""赵老哥""方新侠"等为代表的顶级游资席位。特征是借通道优势在涨停板上扫货封板，擅长打首板、打热点龙头，资金体量大、对人气票有定价权。出现在**首板 / 二板的买一**，是较强的做多信号。
- **波段 / 趋势型（中线持有）**：以"章盟主"为代表，资金体量极大、题材嗅觉敏锐，**很少做一日游**，倾向分段建仓、波段持有数日，风控稳。其席位上榜往往意味着该题材有中线资金驻扎，持续性预期更强。
- **低吸型（不追板、吸调整票）**：在个股回调、分歧或炸板时进场，不在涨停价扫货。这类席位出现在**非涨停的放量阴线 / 十字星**时，反而可能是低位承接信号。
- **砸盘 / 一日游型（次日必卖）**："拉萨天团"（拉萨团结路、东环路等营业部）是典型代表，常被理解为散户与短线资金集中通道。这类席位**大额买入后次日高开 / 低开即出**，对行情几乎无持续推动力；若其多个席位同时大额上榜，次日承压概率显著上升。

使用纪律：**风格判断只用于解读概率，不能当作必涨/必跌的确定信号**。游资会换营业部、会反向操作，标签需以近1—3个月的实际行为校准。

### 机构专用席位的含义

榜单上以"机构专用"四字标识的席位，代表公募、私募、保险、社保等机构账户（交易所做匿名归并，不披露具体机构名）。解读要点：

1. **方向与数量**：买方出现机构专用席位，且数量越多、买入金额占当日成交比越大，筹码锁定度越高，**一日游概率越低**，中线资金介入的可能性上升。
2. **机构 + 游资同现（买方）**：可理解为主流资金达成共识，趋势延续概率相对高。
3. **机构专用席位在卖方**：尤其是在股价高位、已连板的情况下，机构卖出是明确的减仓 / 兑现信号，需警惕。
4. **不要神化机构席位**：机构也会做错、做短，且"机构专用"是合并口径，可能是多家机构净额相抵后的结果；高位放量时"机构买、游资卖"的组合尤其要小心利益输送式接盘（见危险信号）。

### 对倒与一日游识别

**对倒**指资金通过自买自卖制造活跃假象、拉抬或派发筹码。龙虎榜上的典型特征：

- **同一营业部同时出现在买方和卖方榜单**，且买卖金额相近。判断条件：该席位买入额与卖出额比值在 0.7—1.3 之间、且双向金额都较大时，对倒嫌疑高。
- 全榜**买卖席位高度重叠**（多家营业部既买又卖），成交放巨量但榜单合计净买入接近 0，说明是**存量资金互相倒手**，没有真实增量进场。

**一日游**指资金当天上榜买入、次日即了结，对趋势无推动：

- 识别信号：买入席位以一日游风格游资 / 散户通道（如拉萨系）为主；**净买入占成交比偏低**（如 < 3%）却制造了涨停；次日竞价高开幅度小或直接低开。
- 历史回溯法最有效：把当晚的买一买二席位，调取其**最近10—20次上榜后的次日及3日表现**，统计其"次日溢价率""持有3日胜率"，量化该席位是接力盘还是一日游盘。

### 净买入额与封单的关系

打板战法里，**封单**（涨停价上的挂买量）和**龙虎榜净买入**反映的是两个时点、两件事，必须分开看：

1. **封单是盘中实时的承接强度**，反映当下封板的稳固度（封流比 = 封单金额 ÷ 流通市值，越高越稳）；**净买入是盘后结算的资金去向**，反映谁在真买。
2. **强信号组合**：大封单 + 知名打板席位大额净买入 + 榜单合计净买入占成交比高 → 筹码锁定好、次日溢价预期强。
3. **危险背离**：盘中封单看似很大，但盘后榜单显示**净买入为负或对倒严重** → 封单多为对倒撑出来的假封单，次日炸板 / 低开风险高。
4. **一字 / T字板**特别提示：一字板换手极低时往往不触发龙虎榜或榜单金额很小，此时**无榜单≠无风险**，更多要看封单 / 撤单动态而非席位数据。

### 上榜对次日走势的指示意义

上榜本身是中性的——它是"放量异动"的客观结果，方向要靠席位结构判断。归纳为可执行的分级：

- **偏强（次日溢价概率higher）**：买盘集中（买一占比高）+ 知名打板 / 波段席位主导 + 机构席位介入 + 榜单净买入占成交比 > 5% + 买卖席位无重叠。
- **偏分歧（需谨慎，仓位减半或观察竞价）**：多席位分散买入、人气足但买卖均放量 + 榜单净买入为小正或接近0。
- **偏弱 / 危险（倾向回避或次日竞价出）**：净买入为负、对倒明显、卖方出现知名出货 / 一日游席位、高位"机构买游资卖"或"游资买机构卖"组合。

强调：**连板高度越高，龙虎榜的"出货含义"权重越大**。低位首板上榜多偏机会，高位3板以上上榜需把"是否在派发"作为第一判断，而非"谁在买"。

### 用龙虎榜判断龙头接力意愿与持续性

针对龙头票，按下面的流程评估资金的接力意愿与持续性：

1. **认人**：确认买一买二是接力型 / 波段型席位，还是一日游 / 砸盘型席位。接力型主导 → 持续性预期高；一日游主导 → 警惕次日断板。
2. **认结构**：判断是**接力**（新资金进场、买卖席位不重叠、净买入为正且占比高）还是**锁仓后派发**（老资金边拉边出、净买入转负）。接力的持续性强，派发后抛压大。
3. **认连续性**：跟踪同一龙头**连续多日的榜单**，看是否有资金"换手不换庄"——前一日买入席位次日仍在、或新强资金接力进场。**核心资金持续净买入且无大额卖出**，是趋势能延续的最强证据。
4. **认题材共振**：同板块多只票同时上榜、且买方席位有交叉重叠，说明资金在做"板块effect"，龙头持续性强于孤立异动。
5. **量化校验**：对关键席位做历史胜率回测（次日溢价率、3日持有胜率、断板后回撤），用数据而非标签下注。

### 危险信号检查清单（出货 / 对倒 / 知名一日游）

次日决策前逐条核对，命中任意一条都应下调仓位或回避：

- **榜单合计净买入为负**：股价涨停 / 大涨但卖盘 > 买盘，主力借势出货。
- **同一营业部买卖双向大额且金额接近**：对倒嫌疑，活跃量为假。
- **全榜买卖席位高度重叠、净买入≈0**：存量倒手，无真实增量。
- **卖方出现知名出货 / 砸盘席位**（如拉萨系多席大额上榜、机构席位在高位卖出）。
- **买方全是一日游风格席位**：次日大概率高开走低或低开，无接力。
- **高位"机构买 + 游资卖"或"游资买 + 机构卖"组合**：警惕借机构 / 游资效应吸引散户高位接盘的利益输送式出货。
- **封单大但盘后净买入弱 / 为负**：假封单，次日炸板风险高。
- **连板高位首次出现巨量卖出席位**：从"做多"切换到"派发"的信号，优先竞价了结。

总体原则：龙虎榜是**事后的资金画像与概率工具，不是买卖指令**。它最大的价值在于排雷（识别出货、对倒、一日游）和给接力 / 持续性判断提供证据，而非提供必胜信号。


## 十三、风险控制与仓位管理

短线游资战法的核心矛盾在于：追板/打板是一类高波动、负偏度的交易——多数情况下赚取小幅溢价，少数情况下承受连续跌停的大额亏损。本章作为全手册仓位与资金管理的权威出处，所有单票上限、回撤熔断口径以此为准；其余各章涉及仓位的数字不得突破本章上限。本章只给可执行的资金管理规则，不重复战法选股逻辑。

### 13.1 制度约束下的风险边界（量化前提）

（交易制度统一口径见《制度与术语基线》一节，本章只列与资金管理直接相关的差异点。）

- **涨跌幅放大效应**：同样一次"买在板上、次日炸板低开"的失误，在 20cm 标的（创业板/科创板 ±20%）上的单日浮亏可达 -20% 至 -24%（涨停板买入后次日跌停），是 10cm 标的（主板 ±10%）的约 2 倍；30cm 标的（北交所 ±30%）放大更甚。本战法原则上回避 ST 标的，ST/\*ST 涨跌幅以切换日前后券商行情实际显示的涨跌停价为准，不纳入常规仓位桶。
- **新股阶段无涨跌幅**：主板/创业板/科创板新股上市前 5 个交易日不设涨跌幅、第 6 日起恢复对应板块限制；北交所新股上市首日不设涨跌幅、次日起 ±30%。无涨跌幅期间仅适用 30%/60% 两档盘中临时停牌（每次 10 分钟）。这一阶段的标的不适用"涨停板=封单保护"的逻辑，单笔波动可达数十个百分点，须单独按高波动桶（13.3）限额。
- **T+1 不可日内纠错**：买入当日不可卖出。这是所有规则的底层约束——错误一旦在买点形成，最早只能在次日集合竞价（早盘 9:15–9:25，其中 9:15–9:20 可撤单、9:20–9:25 不可撤单，9:25 撮合产生开盘价、9:25–9:30 只申报不撮合）或开盘后处置，期间承受隔夜全部风险。因此短线的风控重心必须前移到"买入前的仓位限额"，而非"买入后的止损执行"。
- **极端不可卖出风险**：一字跌停（开盘即封死跌停、仅时间优先撮合）时散户排队基本无法成交；面值退市（连续 20 个交易日收盘价低于 1 元）、市值退市（主板连续 20 个交易日收盘总市值低于 5 亿元）、突发业绩地雷/立案/重组失败导致的连续一字跌停、长期停牌后复牌的一次性补跌，都会让"止损线"失效。对这类标的，唯一有效的风控是事前排除，而非事中止损。

### 13.2 总仓位与单票上限

以单一账户净值为基准（下称"本金"），按下列硬约束执行，任一条触发即不得加仓：

1. **总仓位上限**：以情绪周期定档，而非固定值；总仓位随情绪周期调节，退潮/冰点空仓。
   - 退潮期/冰点（市场连板高度断裂、炸板率 >50%）：空仓，仅留观察仓。
   - 修复期/启动期（高度重新打出、赚钱效应回升）：总仓位 ≤ 60%。
   - 主升/高潮期（普涨、连板梯队完整）：总仓位 ≤ 90%，且高潮当日及次日不新开仓，只持有与减仓。
2. **单票上限（占本金，按板幅；全章统一口径）**：
   - 10cm 标的（主板老股）：单票 ≤ 20% 本金。
   - 20cm 标的（创业板/科创板）：单票 ≤ 10% 本金。
   - 30cm 标的（北交所）与无涨跌幅新股阶段：单票 ≤ 7% 本金。
   - 单票上限按波动率倒挂设置，目的是让"单票次日跌停"造成的本金回撤近似相等（约 -2% 本金，见 13.3）。
3. **首板试仓与连板加仓分离**：首板/低位首次介入以试仓位（单票上限的 1/2）进入；只有在标的兑现溢价、且情绪周期未恶化时，才允许在次日加至单票上限。不允许在亏损的票上向下加仓（不补仓、不摊低成本）。
4. **同题材集中度**：同一题材/同一连板梯队的合计仓位 ≤ 40% 本金。打板的隐性风险是"看似分散的多只票，实为同一情绪共振"，一旦情绪退潮会同步跌停，分散失效。

### 13.3 单笔最大亏损与回撤的量化口径

把"最大亏损"用本金百分比锁死，而非用持仓百分比，是仓位管理的关键。

- **单笔最大亏损（R）**：以单笔交易亏损 ≤ 2% 本金为基准设定仓位，与单票上限互为校准：
  - 10cm 标的最坏次日跌停 -10%，单票 20% → 本金回撤约 -2%。
  - 20cm 标的最坏次日跌停 -20%（板上买入到次日跌停最坏含溢价回吐可达 -24%），单票 10% → 本金回撤约 -2% 至 -2.4%。
  - 30cm 标的与无涨跌幅新股波动更大，单票压到 ≤ 7% → 本金回撤控制在约 -2% 一线。
  - 据此，本金回撤的"单笔尺子"R 取 2%（对应单笔账户最大风险约 2%）。所有仓位上限以"单票次日跌停 ≤ 1R"为校准目标。
- **单日最大回撤**：当日浮亏（已实现+未实现）达到 -4%（即 2R）本金时，当日剩余时间不再开任何新仓；达到 -6%（3R）时，次日开盘集合竞价无条件减至半仓以下。
- **单周最大回撤**：自然周内本金回撤达 -8%（4R）时，下一交易日起总仓位上限下调一档（如从 60% 降至空仓观察）并维持至周末。
- **月度熔断**：当月本金回撤达 -12%（6R）时，强制空仓观察至少 3 个交易日，仅复盘不交易（详见 13.6 降频与停手机制）。

上述阈值之间满足 1R→2R→3R→4R→6R 的递进关系，便于在盘中快速判断当前处于哪一级风控状态，无需临场计算。

### 13.4 连板加速段的回撤风险与减仓触发

连板的后段（高度板、加速段）是溢价最高、也是回撤最陡的区域，必须用机械规则减仓，不靠盘感。

- **高度衰减信号（满足任一即次日不参与新开、持仓减半）**：
  1. 市场最高连板高度由 N 连板回落到 N-1 且无新高度补上（梯队断层）。
  2. 昨日涨停今日炸板的"反包失败"比例上升、主线/板块连板晋级率（连续 2 日 ≥50% 确认延续）跌破 50% 并连续 2 日下降。
  3. 加速段标的开始出现放量长上影、分时反复破板（盘中开板次数 ≥ 3）。
- **加速段持仓的离场规则**：
  - 持有 3 板及以上的加速票，次日竞价高开 ≥ 5%（10cm）/≥ 8%（20cm）且未一字，开盘后 9:30–9:35 不能再创分时新高的，减仓 1/2 以上。
  - 盘中跌破前一日实体均价（或分时黄线）不收回，止盈剩余仓位，不等跌停。
  - 加速段的核心心法：在板上止盈，而非在炸板后止损。因为 T+1 下"炸板再跑"往往要承受次日低开。

### 13.5 打板/炸板的概率与盈亏期望（客观评估）

公开可核的统计与从业经验给出的区间是：A股全市场无差别打首板的封板成功（不炸板）概率显著低于五成；只有在龙头、低位首板、强情绪周期等强约束下，封板持续性与次日溢价才明显改善。据此做期望管理：

- **不要把打板当作高胜率策略**。无筛选打板的次日溢价期望可能为负（炸板低开+次日竞价分歧）。正期望只来自严格的标的与时机筛选（龙头属性、低位、题材发酵、情绪周期向上）。
- **期望的数学要求**：设单笔止盈幅度约为 +5%~+8%（次日竞价兑现），单笔止损约为 -1R（本金 2%）。即便胜率仅 45%，只要平均盈亏比 ≥ 1.5:1，长期期望仍为正；反之，盈亏比若被"扛单、不止损"压到 1:1 以下，再高的短期胜率也会被一两次连续跌停清零。
- **实操检查**：每周统计自己的"封板率、次日溢价均值、单笔盈亏比、最大单笔亏损"，若连续 4 周次日溢价均值为负或盈亏比 < 1，应判定为策略与当前市场不匹配，进入降频。

### 13.6 容错、止损纪律与降频/停手机制

止损在 T+1 下分为"事前限额"和"事后处置"两层，事前限额是主力，事后处置是补救。

- **事前限额（最重要）**：严格执行 13.2、13.3 的单票与总仓位上限，确保任何单笔错误的本金影响 ≤ 1R。这是 T+1 下唯一真正可靠的止损。
- **事后处置（次日纪律）**：
  - 次日竞价：若标的低开破前一日封板逻辑（如连板票低开 ≥ 5% 且竞价持续走低），9:25 前挂集合竞价卖出（注意 9:20–9:25 不可撤单，挂出前须确认）。
  - 盘中破位：跌破买入逻辑位（如打板买点的分时低点、首板的涨停价）不收回，当日清掉，不留隔夜。
  - 绝不向亏损票补仓、绝不"等反弹再卖"。
- **极端风险的事前排除清单（命中任一即不参与）**：
  1. 收盘价低于 2 元、或趋近 1 元面值的低价股（面值退市风险）。
  2. \*ST/退市风险警示、近一年内有立案/问询/商誉与质押高悬的标的。
  3. 财报披露窗口（业绩预告/年报季）前介入业绩不确定标的（业绩地雷+一字跌停风险）。
  4. 有停牌重组预期、长期停牌后即将复牌的标的（复牌一次性补跌）。
  5. 一字板（无量涨停/跌停）不追、不接——买不到溢价、卖不掉风险。
- **降频与停手机制（量化触发）**：
  - 连续亏损：当日连续 2 笔止损，当日停手；连续 3 个交易日净亏损，次日仓位上限减半。
  - 回撤触发：触及单周 -8%（4R）降一档；触及月度 -12%（6R）强制空仓 ≥ 3 个交易日，期间只复盘、记录、回测，不下任何新单。
  - 复出条件：停手后须满足"用模拟或极小仓位（≤ 5% 本金）连续 5 笔达到正期望"方可逐档恢复仓位，不得一次性恢复满仓。

### 13.7 资金管理与最大回撤控制方案（汇总检查清单）

每个交易日开盘前与盘中按下表自检，任一红线触发即降级处置：

- 开盘前
  - [ ] 确认当前情绪周期档位 → 锁定今日总仓位上限（退潮/冰点空仓 / 修复 ≤60% / 主升 ≤90%）。
  - [ ] 计算 1R = 本金 × 2%，写下今日 2R/3R 的本金亏损绝对值。
  - [ ] 核对持仓是否命中 13.6 极端风险排除清单。
- 盘中/买入前
  - [ ] 单票上限：10cm ≤ 20%、20cm ≤ 10%、30cm/新股 ≤ 7%（占本金）。
  - [ ] 同题材合计 ≤ 40%；新开仓后总仓位不超档位上限。
  - [ ] 单票"次日跌停 ≤ 1R"校验通过。
- 风控红线（触发即执行）
  - [ ] 当日 -4%（2R）→ 停止新开仓；-6%（3R）→ 次日减半。
  - [ ] 当周 -8%（4R）→ 仓位降一档；当月 -12%（6R）→ 强制空仓 ≥ 3 日。
  - [ ] 当日连续 2 笔止损 → 当日停手；连续 3 日净亏 → 仓位减半。
- 收盘后
  - [ ] 记录每笔封板率、次日溢价、盈亏比、最大单笔亏损。
  - [ ] 周度复盘：连续 4 周溢价均值为负或盈亏比 < 1 → 进入降频。

核心原则一句话概括：**用事前的仓位限额把每一次错误锁定在 1R 以内，用递进式回撤熔断防止单日/单周/单月的连续亏损扩大，用降频与停手机制熬过策略与市场不匹配的退潮期。** 短线游资战法的长期收益来自"少亏"，而非"多对"。


## 十四、操作清单与交易禁忌

> 本章是全手册的执行落地层，不重复战法原理，只给可勾选、可量化的检查项与硬约束。所有阈值为默认建议值，需结合个人账户体量与风险偏好校准；标注「★」者为不可妥协的红线项，任一不满足即放弃当笔交易。
>
> （交易制度统一口径见《制度与术语基线》一节，本章只列差异点）下单前的具体涨跌停限价，须以券商行情显示为准；尤其注意 ST 风险警示股自 2026-07-06 起主板由 ±5% 调整为 ±10%，切换日前后以行情实际涨跌停价为准，本战法原则上回避 ST 标的。

### 14.1 盘前检查清单（开盘前完成）

#### A. 情绪研判（判断当下处于情绪周期哪一阶段）
- [ ] 昨日**连板高度**（最高板数）是多少？较前一日是升高、持平还是断板？
- [ ] 昨日**涨停家数 / 跌停家数**、炸板率是多少？炸板率 >30% 视为分歧加剧、>50% 退潮确认。
- [ ] 昨日**晋级率**如何？全市场「首进二」晋级率 >40% 偏健康、<25% 衰竭；主线/板块连板晋级率连续 2 日 ≥50% 确认延续、跌破 50% 分化（两者分母不同，不可混用）。
- [ ] 昨日**最高板与首板**的赚钱效应：打板隔日是否普遍溢价？亏钱效应出现即降仓。
- [ ] 综合定位：当前属于**冰点 / 修复 / 发酵 / 高潮 / 退潮**哪一阶段？★ 退潮期与高潮末期不做接力。
- [ ] 大盘环境：指数是否在关键均线上方？是否处于普跌或缩量阴跌？逆势日只做日内或空仓。

#### B. 题材梳理
- [ ] 昨日**领涨板块**是哪个？是新题材首爆，还是已发酵 ≥3 日的老题材？
- [ ] 题材**级别判断**：政策级 / 事件级 / 业绩级 / 概念级；级别越高、空间越大、持续越久。
- [ ] 题材是否有**消息面催化与持续性**（政策落地、产业订单、龙头业绩），还是纯情绪？
- [ ] 板块内**梯队是否完整**（龙头—跟风—补涨成形），还是只有孤零零一两只？孤股不接力。

#### C. 标的池构建（分层，不超过 8 只重点跟踪）
- [ ] **龙头池**：当前题材最高板 / 人气最高 / 成交额最大的 1–2 只。
- [ ] **接力池**：弱转强、低位首板、二板定龙头候选，标注买点触发条件。
- [ ] **观察池**：有潜在催化但今日不操作，仅记录。
- [ ] 每只标的标注：**板块地位、连板数、流通市值、昨日成交额、关键价位（前高/前低/打板价）**。
- [ ] ★ 剔除：基本面有重大瑕疵（业绩爆雷、立案、退市风险）、近期高位天量放出、纯杂毛（无逻辑跟风、非板块龙头的高位补涨股）。

#### D. 竞价预案（9:15–9:25 动态跟踪，每只标的预设三套脚本）
- [ ] 对每只重点标的写明：**「竞价高开 X% 以上→执行 A 方案；平开/低开小幅→B 方案；低开 >Y% 或大幅杀跌→放弃」**。
- [ ] 关注 9:15–9:20 竞价**撤单变化**（可撤单段虚假挂单多），以 9:25 最终竞价定生效价。
- [ ] 龙头标的：竞价**高开幅度、竞价量能、是否一字/换手**是核心定档信号。
- [ ] ★ 设定当日**最大可用仓位上限**与**单票上限**（10cm 标的 ≤20%、20cm 标的 ≤10%、30cm 标的 ≤7%），写下来，开盘后不临时上调。
- [ ] 设定当日**最大可承受亏损额**（建议账户的 1%–3%），触及即当日停手。

---

### 14.2 盘中检查清单

#### A. 买入触发（满足才动手，不满足只看不做）
- [ ] **打板买入**：标的为板块龙头或卡位股，封板量能充足、分时无大幅杀跌、板上承接 > 撤单；★ 非龙头杂毛不打板。
- [ ] **半路 / 加速买入**：放量突破前高或关键平台，量价配合，板块同步走强（非独自表演）。
- [ ] **回封 / 弱转强买入**：昨日强势股今日低开后分时走强、放量翻红、率先反包，确认资金回流。
- [ ] **分歧转一致**：高位股大幅低开或炸板后被重新拉起、再度封板，确认有新接力资金。
- [ ] ★ 每一笔买入前自问：**逻辑是什么？买点是什么？止损位在哪？目标位在哪？** 四问有一问答不出 → 不买。

#### B. 加 / 减仓
- [ ] **加仓**：仅在浮盈且逻辑被进一步验证（如顺利晋级、板块扩散）时小幅加，★ 严禁亏损补仓 / 越跌越买。
- [ ] **减仓**：冲高遇明显抛压、板块退潮、个股走弱于板块、达到预设减仓位时，先减后看。
- [ ] 单票仓位始终 ≤ 预设上限（10cm ≤20%、20cm ≤10%、30cm ≤7%）；★ 任何时候不满仓单一标的。

#### C. 卖出触发（任一触发即执行，不犹豫、不找理由扛单）
- [ ] **止损**：跌破买入时设定的止损位（如打板买入次日不及预期 / 跌破分时关键支撑 / 板块集体走弱）。
- [ ] **止盈**：达到目标涨幅、冲高乏力、上影线放量、龙头断板、晋级失败、尾盘走弱。
- [ ] **逻辑破坏**：题材证伪、利好兑现成利空、板块退潮、个股掉队，无论盈亏立即了结。
- [ ] **时间止损**：买入后在合理时间窗口内未按预期启动，资金效率低 → 换股或离场。
- [ ] ★ 留意 T+1：当日买入次日才能卖；高位接力的隔日不确定性是最大风险，仓位据此控制。

---

### 14.3 盘后检查清单

#### A. 复盘
- [ ] 更新**情绪周期定位**：今日连板高度、晋级率、炸板率、涨跌停家数较昨日变化，判断明日方向（升温/退潮）。
- [ ] 梳理**今日领涨题材与梯队**：龙头是谁、有无新题材接棒、老题材是否高潮见顶。
- [ ] 逐笔记录**自己的交易**：买卖点、是否按计划执行、盈亏、对错归因（是逻辑错还是执行错）。
- [ ] 标记**违纪交易**（追高杂毛、补仓、超仓、情绪化下单等），单独登记，统计本周违纪次数。

#### B. 龙虎榜
- [ ] 查看重点标的龙虎榜：**席位结构**（知名游资 / 机构 / 量化 / 散户、拉萨系通道）、买卖力量对比、是否有接力资金进场或主力出货。
- [ ] 关注主板上榜触发口径（如收盘涨跌幅偏离值 ±7%、振幅 15%、换手率 20% 等前五），结合席位判断龙头是否仍被资金认可。
- [ ] 机构 / 知名游资**净买入且无明显对倒**→ 加分；**席位大幅净卖出 / 一日游游资接力**→ 警惕次日高开走弱。

#### C. 次日计划
- [ ] 据情绪定位决定**明日总仓位区间**（退潮日降仓 / 空仓，发酵日可进攻）。
- [ ] 列出**明日标的池**与每只的竞价预案、买点、止损位（即回到 14.1 流程）。
- [ ] 预设**明日最大亏损额**与**纪律提醒**（针对今日出现的违纪项重点防范）。

### 14.4 一页式「买入条件清单」（全部满足才下单）

- [ ] ★ 当前情绪周期允许进攻（非高潮末期、非退潮期）。
- [ ] ★ 标的是板块**龙头或核心卡位**，不是杂毛、不是孤股、不是高位补涨接盘。
- [ ] ★ 有清晰**题材逻辑与级别**，且板块整体走强（同步共振，非独自表演）。
- [ ] ★ 有明确**买点触发信号**（打板 / 半路突破 / 弱转强 / 分歧转一致之一）已成立。
- [ ] 量价配合：放量上攻、封板承接充足 / 突破有量，无放天量出货迹象。
- [ ] ★ 已写下**止损位、目标位、计划仓位**，且仓位 ≤ 单票上限（10cm ≤20%、20cm ≤10%、30cm ≤7%）、总仓 ≤ 当日上限。
- [ ] ★ 顺势、有计划、非情绪化、非听消息接盘。
- [ ] 单笔最大潜在亏损 ≤ 当日 / 账户可承受额。

> 任一「★」项不满足 → 放弃本笔，宁可错过，不可做错。

### 14.5 一页式「卖出条件清单」（任一触发即卖，不需全满足）

- [ ] 触及**预设止损位**（买入即设，不下移）。
- [ ] 达到**预设止盈目标**或冲高明显乏力（长上影、放量滞涨）。
- [ ] **逻辑 / 题材被破坏**：证伪、利好出尽、板块退潮。
- [ ] **龙头断板 / 晋级失败 / 个股掉队**于板块。
- [ ] **情绪周期转向退潮**（高度坍塌、晋级率骤降、亏钱效应蔓延）。
- [ ] **龙虎榜显示主力 / 接力资金离场**。
- [ ] **时间止损**：未在合理窗口内按预期启动。
- [ ] 分时出现**放量跳水 / 跌破关键支撑 / 大单砸盘**。

> 卖出只需「任一成立」即执行；扛单、补仓、等反弹回本都是放大亏损的开端。

### 14.6 交易禁忌（红线，违反即记违纪并复盘）

- [ ] ★ **追高杂毛**：买入非龙头、无逻辑的高位跟风 / 补涨股，幻想跟着喝汤。
- [ ] ★ **亏损补仓 / 越跌越买**：摊低成本式补仓，把短线做成被动长线。
- [ ] ★ **满仓单票**：单一标的押满账户，一字跌停 / T+1 无法出货时致命。
- [ ] ★ **逆势打板**：退潮期、高潮末期、大盘普跌日强行接力高位板。
- [ ] ★ **无计划交易**：没有买点、止损、目标就下单；盘中临时起意追单。
- [ ] ★ **情绪化频繁操作**：连续止损后报复性交易、频繁切换、当日反复进出同一票。
- [ ] ★ **听消息接盘**：凭小道消息 / 荐股 / 群里喊单买入，自己说不清逻辑。
- [ ] **不止损 / 移动止损位**：触及止损后下移止损线、找理由扛单。
- [ ] **盈利单急着跑、亏损单死扛**：赚小钱亏大钱，盈亏比倒挂。
- [ ] **临时上调仓位上限 / 亏损额上限**：突破盘前自定的纪律红线。

> 特定提示：本章为执行清单，单票仓位上限按板幅分档（10cm ≤20%、20cm ≤10%、30cm ≤7%，对应单笔账户最大风险约 2%），任何盘中临时上调都属违纪——清单的价值在于把仓位红线钉死在下单之前，而非事后补救。
