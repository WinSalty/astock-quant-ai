### 修复概要（五阶段，每阶段经独立审计 agent 对抗复审）
- **阶段A 资金安全与状态可恢复**：写队列 fail-closed（`flush_confirm` 区分已 drain 与已持久 + on_failure 停开新仓）；持仓状态机/台账/TTL/下单计数器崩溃重启可从 QMT 权威 + 本机台账重建（原纯内存、重启即裸奔扛单）；卖出状态机死锁复位（同步失败/拒单/全撤零成交/挂不到价超时四类 revert + 卖单 TTL）；再买入命中 SOLD 开新单元；回调改「内存权威先于 qmt 镜像」；持仓单元加锁防迭代崩溃。
- **阶段B 风控护栏与价位口径真正生效**：行情断流/下单断线/账户回撤/单票止损/限价偏离五道闸从死代码改为真生效（fail-closed 不再 fail-open）；ST 主板涨停 5%、signal_close≤0→MISSING、透传 `name` 识别 ST；对账检出真异常→持久标记阻断次日开仓（SELL_ONLY 至人工清除）+ 触发补采。
- **阶段C 回测裁决与龙头打分可信**：go/no-go 改净收益口径（强制含费）+ 统计显著性，卖出侧防涨停封板顺延、未平仓样本保守 mark 计入；龙头强度分位/分布/最高板基准改由**全涨停池**构建（复审修复了「全池接线断裂、生产路径不生效」的 P0）。
- **阶段D 供数与建仓闭环**：最强龙头（秒封一字/竞价强开）在默认配置下有可执行成交路径（定盘段转 OPENING + 解锁幂等 + 开盘前 TTL 顺延）；半路首板不误路由竞价强开；`first_board_vol` float→int 落表（原恒 NULL、量能比因子结构性降级）；行情取数前先订阅。
- **阶段E 复盘/回流/并发/鉴权**：回流时间 tz-aware→UTC naive（不再错 8h）；当日盈亏差分仅上一交易日成立（不把多日当单日）；回流接口加来源 IP 白名单；COALESCE 防空覆盖扩到所有可空列；写连接 busy_timeout；竞价轮询单轮异常不放弃整段。
